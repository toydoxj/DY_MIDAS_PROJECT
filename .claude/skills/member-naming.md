# 기본사항

## 층표기 
없는 경우 : 일반적으로 전체 층을 의미함.
단일층 : 숫자만 표기하는 것이 일반적  / 예: 8F -> 8, B4F -> -4 or B4 (-4로 표기하는 경우가 많음)
여러층을 묶을 수도 있음 / 예 : 4층 5층, 6층 등 연속되는 층에 설치되는 경우 -> 4~6, 2층 8층 등 불연속 층인 경우 -> 2,8
R : 지붕층
PH : 옥탑층
PHR : 옥탑지붕층
P : Pit 층
M : 중간층(메자닌)

## 번호
번호는 1,2,3,11 등등 이루어지나 1A, 5B 등 A,B,C 등으로 구분하기도 함.

# 부재명 명명 규칙

## 보 (Beam)
- 형식 : {층표기}{구분자}{번호}
- 예시 : RB1, 8G2, 2~4G3A, -1B8E, B2, WG1
- 구분자 
  G : Girder (기둥과 기둥을 연결 경우에 따라서 지진력 저항시스템에 따른 상세 적용 필요함)
  B : Beam (Girder와 연결되는 부재로 대부분 지진력 저항시스템의 영향을 받지 않음)
  SG 또는 SB : Girder와 Beam 중 강구조를 의미함.
  SRG 또는 SRB : 매립형합성보 의미
  TG 또는 TB : 전이보를 의미함 (ALL 배근 적용), 최하층의 경우 지중보(Tie Girder를 의미하
  경우도 있음)
  WG : Wall Girder (벽체와 연결되는 Girder로 대부분 힘이 작음, ALL 배근 적용, 부재력이 과다하게 큰 경우 검토 필요함)
  FG : 지붕보 (Footing Girder)
  LB : 인방보 (Lintel Beam)

## 기둥 (Column)
- 형식 : {층표기}{구분자}{번호}
- 예시 : C1, 3~6C2A, -4~1SC2, -5~-2SRC1C
- 구분자 
  C : Column (일반적인 기둥, 대부분 RC)
  SC : Steel Column
  SRC : 매립형 합성 기둥


## 슬래브 (Slab)

- 형식: `{층표기}{구분자}{번호}`
- 예시: `3S1`, `B2S3`, `RPS1`
- 구분자:
  - `S`: 일반 슬래브
  - `PS`: 플랫 플레이트 또는 포스트텐션 슬래브(프로젝트 규칙에 따라 선택)
- 비고:
  - 슬래브 분할 구역이 필요한 경우 `S1A`, `S1B`처럼 접미 문자를 사용한다.
  - 동일층에서 중복 번호를 피하고 구조도면의 패널명과 매칭 가능해야 한다.

## 벽체 (Wall)

- 형식: `{층표기}{구분자}{번호}`
- 예시: `2W1`, `-1SW2`, `3~5CW1A`
- 구분자:
  - `W`: 일반 RC 벽체
  - `SW`: 전단벽(Shear Wall)
  - `CW`: 코어벽(Core Wall)
- 비고:
  - 벽체 방향 구분이 필요하면 `W1X`, `W1Y` 접미를 허용한다.
  - 내진 상세 대상 벽체는 전단벽 구분자(`SW`)를 우선 사용한다.

## 기초 (Foundation)

- 형식: `{층표기}{구분자}{번호}`
- 예시: `F1`, `BF2`, `PF1A`
- 구분자:
  - `F`: 독립기초 또는 줄기초
  - `BF`: 매트/기초보 계열(프로젝트 규칙에 따라 Foundation Beam 포함)
  - `PF`: 파일캡 또는 말뚝기초 계열
- 비고:
  - 지하층 기초는 층표기를 생략하거나 `B` 접두를 병행할 수 있다.
  - 기초 부재명은 지반/기초 상세도와 1:1 추적 가능해야 한다.
