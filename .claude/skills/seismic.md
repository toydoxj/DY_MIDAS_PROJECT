# 지진하중 검토를 위한 자료 체크

유효지반가속도 : S
단주기 설계스펙트럼 가속도 : Sds = S \* 2.5 \* Fa \* 2/3
1초주기 설계스펙트럼 가속도 : Sd1 = S \* Fv \* 2/3

기반암 깊이가 20m를 초과하고 지반의 평균전단파속도가 360m/s 이상인 경우 fv의 80% 적용
지반분류가 S5이고 기반암의 깊이가 불분명한 경우 Fa,Fv의 110% 적용

단주기 지반증폭 계수 Fa : S값에 따라 아래 표를 이용 (중간값은 직선 보간함)

| S1 | 1.12 | 1.12 | 1.12 |
| S2 | 1.4 | 1.4 | 1.3 |
| S3 | 1.7 | 1.5 | 1.3 |
| S4 | 1.6 | 1.4 | 1.2 |
| S5 | 1.8 | 1.3 | 1.3 |

1초주기 지반증폭 계수 Fv : S값에 따라 아래 표를 이용 (중간값은 직선 보간함)

# 등가정적지진하중 산정

1. 건물의 높이 : 프로젝트 정보에서 가져옴 (해석 높이)
2. 건물의 중요도 : 특, (1), (2), (3) 중 선택

- 중요도 계수 (Ie)

| 특 | (1) | (2) | (3) |
| --- | --- | --- | --- |
| 1.5 | 1.2 | 1.0 | 1.0 |

3. 지진력저항시스템 
4. 지반의 분류
5. 구조물의 중량 : W
6. 근사고유주기 : Ta
7. 주기 상한계수 : Cu
8. 주기 : T
9. 지진 응답계수 : Cs

- Cs = Sds / ( R / Ie ) 
- 상한값 
T<=TL 일때, Cs = Sd1 / ((R/Ie) \* T) 
T>TL 일때, Cs = Sd1 \* TL /  ((R/Ie) \* T^2) 
- 하한값
Cs = 0.044Sds \* Ie >= 0.01
