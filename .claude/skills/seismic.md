# 지진하중 검토를 위한 자료 체크

유효지반가속도 : S
단주기 설계스펙트럼 가속도 : Sds = S \* 2.5 \* Fa \* 2/3
1초주기 설계스펙트럼 가속도 : Sd1 = S \* Fv \* 2/3

## 4.2.1 설계응답스펙트럼의 정의

지진의 설계응답스펙트럼은 다음 식에 따라 구한 후 그림 4.2-1과 같이 작성한다.

1. T = 0 일때, 스펙트럼가속도 S는 식 (4.2-1)에 의한다.
2. T₀ < T ≤ Ts 일때, 스펙트럼가속도 Sa는 4.2.2에 따라 산정되는 Sds와 같다.
3. Ts < T ≤ TL 일때, 스펙트럼가속도 Sa는 식 (4.2-2)에 의한다.
4. T > TL 일때, 스펙트럼가속도 Sa는 식 (4.2-3)에 의한다.

- Sa = 0.6 \* (Sds/T₀) \* T + 0.4Sds ... (4.2-1)
- Sa = Sds/T ... (4.2-2)
- Sa = (Sd1 \* TL) / T² ... (4.2-3)

여기서 T : 구조물의 고유주기(초)
- T₀ = 0.2Sd1 / Sds
- Ts = Sd1 / Sds
- TL = 8 초

기반암 깊이가 20m를 초과하고 지반의 평균전단파속도가 360m/s 이상인 경우 fv의 80% 적용
지반분류가 S5이고 기반암의 깊이가 불분명한 경우 Fa,Fv의 110% 적용
