# RC 보 설계 KDS 기준 요약

출처: 국가건설기준센터(KCSC) — https://www.kcsc.re.kr/standardCode/list/101014

본 폴더는 RC(철근콘크리트) 보(beam) 설계와 관련된 한국건설기준(KDS) 14 20 계열 기준 중에서 **보 설계 절차 및 제한사항**을 실무 참조용으로 정리한 것이다. 본 요약은 설계 기준의 법적 효력을 대체하지 않으며, 시공·구조검토 시에는 반드시 원문을 확인해야 한다.

## RC 보 설계 절차 개괄

아래 흐름은 위 5개 기준을 결합하여 일반적인 RC 보의 설계·검토에서 수행하는 작업 순서다.

```
┌──────────────────────────────────────────────────────────────┐
│ 1. 하중 및 해석                                              │
│   - 고정/활하중, 지진·풍하중 조합 → Mu, Vu, Tu, Pu 산정      │
│   - (KDS 14 20 01 하중조합 / KDS 17 10 00 내진)              │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 2. 단면 가정 및 재료 선정                                    │
│   - b × h, fck, fy                                           │
│   - 최소 단면 제한 (내진 SMF: b ≥ 250mm, b/h ≥ 0.3 등)        │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 3. 휨설계 (KDS 14 20 20)                                     │
│   - 설계가정: ε_cu=0.0033(fck ≤40MPa), 등가응력블록          │
│   - 최소철근량 φMn ≥ 1.2 Mcr (식 4.2-1)                      │
│   - 인장지배단면 여부 확인 (εt ≥ 0.005 or 2.5εy)             │
│   - T형보·깊은보 특별 규정                                    │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 4. 전단·비틀림 설계 (KDS 14 20 22)                           │
│   - φVn = φ(Vc + Vs) ≥ Vu                                     │
│   - 최소 스터럽 & 간격 제한 (d/2, 600mm 등)                  │
│   - 비틀림 고려 여부 판단 → 폐쇄스터럽                        │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 5. 철근 상세 (KDS 14 20 50)                                  │
│   - 피복두께(보 40mm, fck≥40MPa시 -10mm 가능)                │
│   - 순간격: 25mm 이상, db 이상, 1.33×굵은골재                 │
│   - 횡철근 폐쇄 & 표준갈고리                                  │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 6. 정착·이음 (KDS 14 20 52)                                  │
│   - ld (인장), ldc (압축), ldh (표준갈고리 정착)              │
│   - 연속철근 1/4 or 1/3 받침부 연장                           │
│   - 겹침이음 A/B급, 150mm 이상                                │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 7. 내진상세 (KDS 14 20 80) — 지진하중 적용 시                │
│   - IMF(중간모멘트골조) or SMF(특수모멘트골조) 선택          │
│   - 양단 2h 구간 후프철근 배치                                │
│   - 후프간격: d/4, 8db, 24dhp, 300mm 중 최소                  │
│   - 설계전단력은 소성힌지기반 (Mpr) 또는 2E 적용             │
└──────────────────────────────────────────────────────────────┘
```

## 참고

- 원문 텍스트는 `../raw/` 아래에 각 코드별로 저장되어 있다.
- 각 요약 문서 말미에는 원문 섹션 번호(예: `4.2.2(1)`)를 병기하여 추적이 용이하도록 했다.
- 기호(Mn, Vc 등)의 정의는 원문 1.5절을 참조한다.
