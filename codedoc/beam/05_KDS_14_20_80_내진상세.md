# KDS 14 20 80 콘크리트 내진설계기준 (2021-02-18)

> RC 보의 내진설계 상세(IMF·SMF)를 발췌 정리. 원문 `../raw/KDS_14_20_80_raw.txt`.

## 0. 지진 시스템 매트릭스 (4.1.1)

| 구분 | 대상 | 설계 기준 장 |
| ---- | ---- | ----------- |
| 보통모멘트골조(OMF) | 약진지역 / 낮은 지진위험도 | KDS 14 20 01~72 (일반) + 4.1 |
| 중간모멘트골조(IMF) / 중간구조벽 | 중진지역 / 중간 지진위험도 | **4.3** + 4.1 |
| 특수모멘트골조(SMF) / 특수구조벽 | 강진지역 / 높은 지진위험도 | **4.4 ~ 4.9** + 4.1 |
| 지진력에 저항하지 않는 부재 | 횡력 저항 제외 가정 부재 | 4.2 |

## 1. 특기 사항 (4.1)

### 재료 (4.1.4, 4.1.5) — SMF & 특수 구조벽
- 콘크리트: **fck ≥ 21 MPa**.
- 경량콘크리트: **fck ≤ 35 MPa** (실험으로 입증 시 초과 가능).
- 주철근: KS D 3504 **S등급 특수내진용** 또는 일반구조용으로 다음 두 조건 만족:
  - 실제 fy − 공칭 fy **≤ 120 MPa**
  - 실제 fu / 실제 fy **≥ 1.25**
- **주철근 fy ≤ 600 MPa**
- 전단철근 fy: 선부재 ≤ 500 MPa, 벽체 ≤ 600 MPa

### 기계적이음·용접이음 (4.1.6, 4.1.7)
- **유형 1** 기계적이음: 기둥/보 단부로부터 또는 항복이 예상되는 단면부터 **부재 깊이의 2배 이내에서 사용 불가**.
- **유형 2** 기계적이음: 이음철근 규정 인장강도(fu) 발휘 가능, 위치 제한 없음.
- 용접이음: 위 동일 구간(2h) 사용 불가, 스터럽·띠철근을 종방향 철근에 용접 금지.

## 2. 중간모멘트골조(IMF) 보 (4.3.4)

| 항목 | 요구사항 |
| ---- | -------- |
| 휨강도 비율 | 접합면 **정휨강도 ≥ 부휨강도 × 1/3**. 부재 어느 위치에서나 정 또는 부휨강도 ≥ 양단 접합면 최대 휨강도 × 1/5. |
| **후프 구간** | 보 양단에서 지지부재 내측면부터 **2·h 길이 구간**에 후프철근 배치. 첫 후프는 지지부재면 **50mm 이내**. |
| **후프 최대 간격** | `min(d/4, 8·db(최소 종방향철근 지름), 24·dhp(후프철근 지름), 300mm)` |
| **스터럽 간격** | 부재 전 길이에 걸쳐 **s ≤ d/2** 이하 |

### IMF 설계 전단력 (4.3.3)

다음 중 **작은 값** 이상 사용:
- (2) 순경간 각 고정단 공칭휨강도(Mn)에 따른 전단력 + 계수중력하중 전단력
- (3) 내진설계 하중조합에서 **지진하중 2배(2E)**로 계산한 최대 전단력

## 3. 특수모멘트골조(SMF) 휨부재 (4.4)

### 3.1 기하 제한 (4.4.1(2))

- **Pu ≤ (1/10)·fck·Ag**
- **순경간 ln ≥ 4·d**
- **b/h ≥ 0.3**
- **b ≥ 250 mm**
- **b ≤ 지지부재 폭 + 3h/4 × 2** (양측 합)

### 3.2 축방향(휨) 철근 (4.4.2)

- 상/하부 어느 단면에서도 **ρ ≥ max(0.25·√fck/fy, 1.4/fy)** 이면서 KDS 14 20 20(식 4.2-1)의 최소량 이상.
- **ρ ≤ 0.025**
- 상·하부에 **최소 연속 2개 이상** 철근.
- 접합면 **정모멘트강도 ≥ 부모멘트강도 × 1/2**, 부재 어느 위치에서나 정 또는 부 강도 ≥ 양단 최대 휨강도 × 1/4.
- **겹침이음**: 이음 구간에 후프/나선철근이 있어야 함. 횡방향 철근 간격 **≤ min(d/4, 100mm)**. 사용 금지 위치:
  - 접합부 내부
  - 접합면부터 **2·h 이내**
  - 비탄성 횡변위에 의한 휨항복이 일어나는 곳

### 3.3 횡방향 철근(후프철근) (4.4.3)

**후프 필수 구간**:
- ① 양단 받침부면부터 **2·h 구간**
- ② 비탄성 횡변위로 휨항복 가능한 단면 좌우로 **각 2·h 이상** 구간

**배치**:
- 첫 후프는 지지부재면 **50 mm 이내**.
- **최대 간격 s**: `min(d/4, 8·db(축방향 최소 지름), 24·dhp(후프 지름), 300mm)`
- 후프로 감싸는 축방향 철근은 KDS 14 20 50(4.4.2(3)③) 횡방향 지지.
- 후프 외 구간: 부재 전 길이에 걸쳐 **s ≤ d/2** 이내 간격으로 양단 **내진갈고리** 갖춘 스터럽.
- 후프철근은 **스터럽(양단 내진갈고리) + 연결철근**으로 구성 가능. 연결철근은 한쪽은 135°, 반대쪽은 90° 갈고리로 번갈아 배치.

### 3.4 전단강도 (4.4.4)

**설계전단력 Ve**:
- 접합면 사이 부재 구간에 대해 **단부에서 예상 휨강도 Mpr의 정·부 조합**과 경간을 따르는 **계수중력하중**에 의한 전단력으로 결정.
- Mpr 계산 시 fy 대신 **1.25·fy** 사용 (4.5.2 참조).

**Vc = 0 가정 조건**:
- 지진하중에 의한 전단력이 해당 구간 최대 소요전단강도의 **1/2 이상**이고
- 계수축력이 **Pu ≤ Ag·fck/20** 이하일 때.

## 4. 특수모멘트골조 접합부 (4.6)

### 4.6.1 기본
- 접합면 보 종방향 철근 인장응력을 **1.25·fy**로 가정해 작용력 결정.
- 보 종방향 철근이 보-기둥 접합부를 통과 시, 보 종방향 철근에 평행한 기둥 변 길이 h_c가 식 (4.6-1)/(4.6-2) 만족 필요.

### 4.6.2 접합부 횡보강
- 구조 부재로 구속되지 않은 접합부에는 **4.5.4 규정의 후프철근** 배치.
- 접합부 **4면 모두 보**가 연결(각 보 폭 ≥ 기둥폭 3/4)인 경우, 가장 작은 깊이의 보 깊이 구간 내에서 요구 후프량의 **1/2**만 배치 가능 — 간격은 **150 mm까지** 증가 가능.

### 4.6.3 접합부 전단강도

보통중량골재 콘크리트:
- 4면 구속: **Vn ≤ 1.7·√fck·bj·hc**
- 3면 또는 반대방향 2면 구속: **Vn ≤ 1.25·√fck·bj·hc**
- 기타: **Vn ≤ 1.0·√fck·bj·hc**
- 경량콘크리트: 위 값의 **3/4**.

(각 접합면의 3/4이 인입부재와 접합되어 있으면 구속된 것으로 간주)

### 4.6.4 인장철근 정착 (보통중량, 90°표준갈고리)
- **ldh ≥ max(8·db, 150 mm, 식 (4.6-3) 값)**
  - 식 (4.6-3): `ldh = fy·db / (5.4·√fck)` (단, 단위 mm/MPa)
- 경량콘크리트: `8·db` 이상, 190mm 이상, 식 (4.6-3)의 **1.25배** 이상.
- 직선 철근 정착길이 **ldh × 2.5 이상** (콘크리트 1회 타설 깊이 ≤ 300 mm),
  깊이 > 300 mm이면 **3.5배 이상**.
- 횡구속 심부에 걸리지 않는 부분의 매입정착길이는 **1.6배 증가**.
- 에폭시 도막 철근은 KDS 14 20 52(4.1.2) 계수 적용.

## 5. 지진력에 저항하지 않는 부재 (4.2)

### 4.2.2 설계변위 단면력 ≤ 설계강도
- 하중조합 1.2D + 1.0E + 0.2(or 0.9) 위험값 사용, 창고 제외 L > 5 kN/m² 시 하중계수 0.5 가능.
- Pu ≤ (1/10)fck·Ag: **4.4.2(1) + 스터럽 s ≤ d/2** 이하.
- Pu > (1/10)fck·Ag: **4.5.3, 4.5.4(1)③, 4.5.4(3), 4.5.5** 만족. 띠철근 간격 so = min(6·db(최소 종방향), 150mm).
- Pu > 0.35·Po: 추가 (3) + 횡방향 철근량 **1/2 이상**, 전 기둥 간격 유지.

### 4.2.3 설계변위 단면력 > 설계강도
- 재료 4.1.4, 4.1.5 / 기계·용접이음 4.1.6, 4.1.7.
- Pu ≤ (1/10)fck·Ag: **4.4.2(1) + 4.4.4** 만족, 스터럽 s ≤ d/2.
- Pu > (1/10)fck·Ag: **4.5.3(1), 4.5.4, 4.5.5, 4.6.2(1)** 만족.

## 6. 보 내진설계 체크리스트

### 공통 (SMF)
- [ ] fck ≥ 21 MPa, 주철근 S등급 또는 조건부 일반철근
- [ ] 기계적/용접이음 2h 제한 확인

### SMF 보 기하
- [ ] Pu ≤ 0.1·fck·Ag
- [ ] ln ≥ 4·d, b/h ≥ 0.3, b ≥ 250 mm

### SMF 보 축방향
- [ ] 0.25√fck/fy 및 1.4/fy ≤ ρ ≤ 0.025
- [ ] 상·하부 연속 철근 ≥ 2개
- [ ] 정모멘트강도 ≥ 부모멘트강도 × 1/2
- [ ] 2·h 구간, 접합부 내, 항복구간에 겹침이음 금지

### SMF 보 횡방향
- [ ] 양단 2·h 구간 후프 배치, 첫 후프 ≤ 50 mm
- [ ] 후프 간격 ≤ min(d/4, 8·db, 24·dhp, 300 mm)
- [ ] 후프 외 구간 스터럽 간격 ≤ d/2
- [ ] 양단 내진갈고리 스터럽, 연결철근 135/90° 번갈아

### SMF 보 전단
- [ ] Ve는 Mpr(1.25fy 기준) 양단 조합 + 계수중력 기반
- [ ] Vc = 0 조건 검토

### IMF 보
- [ ] 정휨강도 ≥ 부휨강도 × 1/3
- [ ] 양단 2·h 구간 후프, 간격 ≤ min(d/4, 8db, 24dhp, 300mm)
- [ ] 그 외 스터럽 s ≤ d/2
- [ ] 설계전단력: Mn 기반 vs 2E 중 큰 값

### 접합부 (SMF)
- [ ] 보 철근 1.25·fy 가정 작용력
- [ ] 기둥폭 vs 보철근 db 관계식 (4.6-1) 검토
- [ ] 접합부 Vn 한계 (1.7/1.25/1.0 √fck·bj·hc)
- [ ] 90°갈고리 ldh ≥ max(8·db, 150, fy·db/(5.4·√fck))
