# KDS 14 20 50 콘크리트구조 철근상세 설계기준 (2022-01-11)

> 보 철근의 가공, 피복, 간격, 횡철근 상세를 발췌 정리. 원문 `../raw/KDS_14_20_50_raw.txt`.

## 1. 표준갈고리 (4.1.1)

### 주철근
- **180° 표준갈고리**: 반원 끝에서 `4·db 이상`, 또한 `60mm 이상` 연장.
- **90° 표준갈고리**: 구부린 끝에서 `12·db 이상` 연장.

### 스터럽 / 띠철근
- **90° 표준갈고리**
  - D16 이하: 구부린 끝에서 `6·db 이상`
  - D19, D22, D25: 구부린 끝에서 `12·db 이상`
- **135° 표준갈고리** (D25 이하): 구부린 끝에서 `6·db 이상`

## 2. 구부림 최소 내면 반지름 (4.1.2)

### 주철근 (표 4.1-1)
| 철근 크기 | 최소 내면 반지름 |
| --------- | ----------------- |
| D10 ~ D25 | 3·db |
| D29 ~ D35 | 4·db |
| D38 이상  | 5·db |

### 스터럽·띠철근
- D16 이하: **2·db**
- D19 이상: 표 4.1-1에 따름

### 기타
- 모든 철근은 **상온**에서 구부린다 (예외 시 책임구조기술자 승인).
- 콘크리트에 묻힌 철근은 **현장에서 구부리지 않음** (원칙).

## 3. 최소 피복 두께 (4.3)

### 현장치기 콘크리트 (비프리스트레스, 4.3.1)
| 노출조건 / 부재 | 최소 피복 |
| --------------- | --------- |
| 수중 타설 | **100mm** |
| 흙에 접하여 영구히 묻힘 | **75mm** |
| 흙 접촉 또는 옥외 노출 / D19 이상 | **50mm** |
| 흙 접촉 또는 옥외 노출 / D16 이하, φ16 이하 철선 | **40mm** |
| 옥외·흙 접촉 없음 / 슬래브·벽체·장선(D35 초과) | **40mm** |
| 옥외·흙 접촉 없음 / 슬래브·벽체·장선(D35 이하) | **20mm** |
| **옥외·흙 접촉 없음 / 보, 기둥** | **40mm** |
| 쉘·절판부재 | **20mm** |

> **fck ≥ 40MPa**인 경우 보·기둥 피복두께에서 **10mm 저감 가능** (= 30mm).

### 현장치기 PSC (4.3.2)
- 영구 토중: 75mm
- 옥외·흙 접촉 벽체·슬래브: 30mm / 기타: 40mm
- 옥외·흙 접촉 없음 보·기둥: 주철근 40mm, 띠철근/스터럽/나선철근 30mm

### 다발철근 (4.3.4)
- 피복두께 ≥ `min(50mm, 다발 등가지름)`
- 영구 토중 묻힘: ≥ 75mm
- 수중 타설: ≥ 100mm

### 특수 환경 (노출범주 ES, 4.3.6)
| 조건 | 현장치기 | 프리캐스트 |
| ---- | -------- | ---------- |
| 벽체·슬래브 | **50mm** | 40mm |
| 기타 ES1, ES2 | **60mm** | 50mm |
| 기타 ES3 | **70mm** | 50mm |
| 기타 ES4 | **80mm** | 50mm |

## 4. 철근 배치 (4.2)

### 허용오차 (4.2.1, 표 4.2-1)
| 유효깊이 d | 허용범위 | 최소 피복두께 |
| ---------- | -------- | ------------- |
| ≤ 200mm | ±10mm | −10mm |
| > 200mm | ±13mm | −13mm |

- 철근 단부·종방향 구부림: **±50mm** (일반)
- 브래킷·내민받침 불연속단: **±13mm**
- 기타 부재 불연속단: **±25mm**
- 피복두께 오차는 **요구 최소 피복두께의 1/3 이내**.
- 배치위치가 **db 이상** 벗어나려면 책임구조기술자 승인.

### 간격 제한 (4.2.2)
| 항목 | 최소값 |
| ---- | ------ |
| 동일 평면 수평 평행철근 순간격 | **25mm** 이상, **db** 이상 (또한 굵은골재 최대치수의 4/3 이상) |
| 상하 2단 이상 배치 시 수직 순간격 | **25mm** 이상, **상하 동일 연직면** |
| 압축부재(기둥) 축방향 철근 순간격 | **40mm** 이상, **1.5·db** 이상 |
| 벽체·슬래브 휨 주철근 간격 | **3·h** 이하, **450mm 이하** |

### 다발철근 (4.2.2(6))
- 이형철근으로, 묶음은 **최대 4개**. 스터럽·띠철근으로 둘러싸야 함.
- 경간 내에서 끝나는 다발철근 내 개별 철근은 **40·db 이상** 엇갈리게.
- 다발지름 = 등가단면적 환산 단일 철근 지름.
- **보에서 D35 초과 철근은 다발 불가**.

## 5. 보 횡철근 (4.4.1)

| 조항 | 요구사항 |
| ---- | -------- |
| (1) | **보의 압축철근**은 띠철근/스터럽 또는 등가 용접철망으로 둘러싸야 함. 크기·간격은 4.4.2(3) 규정(기둥 띠철근) 준용. 압축철근이 배치된 전 구간 적용. |
| (2) | 받침부에서 응력 반전 또는 **비틀림**을 받는 휨 골조부재의 횡철근은 **폐쇄띠철근**, 폐쇄스터럽 또는 나선철근. |
| (3) | 폐쇄띠/폐쇄스터럽 구성: 한 가닥으로 종방향철근 둘레를 돌려 표준갈고리로 중첩, 또는 **1.3·ld 겹침이음(B급)** 또는 KDS 14 20 52(4.4.4) 정착. |

## 6. 구조 일체성 요구 조건 (4.7)

### 현장치기 보·골조
- 테두리보에 다음 **연속철근**을 기둥 축방향 철근으로 둘러싸인 부분을 지나 **전 경간**에 배치:
  - ① 받침부에서 요구되는 **부모멘트 철근의 1/6 이상**, 최소 **2개 이상** 인장철근
  - ② 경간 중앙부 **정모멘트 철근의 1/4 이상**, 최소 **2개 이상** 인장철근
- 불연속 받침부에서 항복강도 발휘 정착(표준갈고리 또는 확대머리 이형철근, KDS 14 20 52).
- (3) 이 연속철근은 KDS 14 20 22(4.5.3) 형식 횡방향철근으로 둘러쌈.
- (4) 상부철근 이음은 **경간 중앙 부근**, 하부철근 이음은 **받침부 부근**에서 **B급 인장겹침이음** (또는 기계적·용접이음).
- (5) 테두리보 외 부재에서 위 (3) 횡방향철근이 없는 경우:
  - 정모멘트 철근의 1/4 이상, 최소 2개가 기둥 축방향 철근에 의해 둘러싸인 부분을 지나야 함.
  - 연속 또는 받침부 주변 B급 이음.
  - 불연속 받침부에서 항복강도 발휘 정착.

## 7. 수축·온도철근 (4.6)

| 항목 | 규정 |
| ---- | ---- |
| 최소 철근비 (fy ≤ 400MPa 이형철근 슬래브) | **0.0020** |
| 최소 철근비 (fy > 400MPa) | `max(0.0015·400/fy, 0.0014)` — 어떤 경우도 0.0014 이상 |
| 상한 | 단위 폭 m당 **1,800 mm²** 이상 배치 불필요 |
| 간격 | 슬래브 두께의 **5배 이하, 450mm 이하** |

## 8. 보 철근 상세 체크리스트

### 가공·피복
- [ ] 스터럽 135° 갈고리 연장 ≥ 6·db (D25 이하)
- [ ] 구부림 내면 반지름: 스터럽 D16 이하 → 2·db
- [ ] 보 피복두께 ≥ 40mm (fck ≥ 40MPa → 30mm 가능), ES 환경 상향

### 배치
- [ ] 수평 순간격 ≥ max(25mm, db, 4/3·굵은골재)
- [ ] 2단 이상 시 수직 순간격 ≥ 25mm, 동일 연직면
- [ ] 다발철근: D35 이하, 4개 이하, 개별철근 40·db 엇갈림

### 횡철근
- [ ] 압축철근 구간에 스터럽/띠철근 폐쇄 배치
- [ ] 비틀림·반전응력 구간은 **폐쇄**스터럽
- [ ] 폐쇄스터럽 이음 B급 (1.3·ld) 겹침이음

### 구조 일체성
- [ ] 테두리보: 상부 부모멘트 철근 1/6 + 하부 정모멘트 철근 1/4 이상 전 경간 연속
- [ ] 불연속 받침부 정착 (표준갈고리/확대머리)
