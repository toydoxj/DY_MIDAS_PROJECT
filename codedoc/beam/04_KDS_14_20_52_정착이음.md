# KDS 14 20 52 콘크리트구조 정착 및 이음 설계기준 (2024-12-30)

> RC 보 철근의 정착길이, 표준갈고리 정착, 휨철근 정착, 이음(겹침·기계적·용접) 규정 발췌 정리. 원문 `../raw/KDS_14_20_52_raw.txt`.

## 1. 정착 일반 (4.1.1)

- 모든 철근은 인장력 또는 압축력이 단면 양측에서 발휘되도록 **묻힘길이·갈고리·기계적 정착** 또는 조합으로 정착한다.
- **갈고리는 압축철근 정착에 유효하지 않다**.

## 2. 인장 이형철근·이형철선 정착 ld (4.1.2)

두 가지 방법 중 선택 — **ld ≥ 300mm**.

### (2) 기본정착길이 × 보정계수

```
ldb = (0.6·db·fy) / (λ·√fck)   (식 4.1-1)
```

보정계수 (표 4.1-1) — 철근 지름 및 배근 조건에 따라 1.0 또는 1.5 적용.

배근 보정계수:
- **α (철근 배치 위치)**
  - 상부철근(정착길이 하부 300mm 초과 신선 콘크리트): **1.3**
  - 기타 철근: **1.0**
- **β (도막계수)**
  - 피복두께 < 3·db 또는 순간격 < 6·db 인 에폭시/아연-에폭시 도막: **1.5**
  - 기타 에폭시/아연-에폭시 도막: **1.2**
  - 아연도금·무도막: **1.0**
  - 상부철근인 에폭시 도막일 때 α·β ≤ 1.7.

### (3) 정밀식 (식 4.1-2)

```
ld = (0.90·db·fy·α·β·γ) / (λ·√fck · (c + Ktr)/db)
```
- **(c + Ktr)/db ≤ 2.5**
- γ (크기계수): D19 이하 = 0.8, D22 이상 = 1.0
- c: 철근 중심에서 콘크리트 표면까지 최단거리 또는 철근 중심 간 거리/2 중 **작은 값**
- Ktr (횡방향 철근지수): 계산 간소화를 위해 **0**으로 가정 가능

### (4) 소요철근량 초과 배치 시 감소

- `(As,req / As,prov)` 을 ld에 곱해 감소 가능. **단, 감소 후 ld ≥ 300mm**, fy 발휘 특별 요구 위치엔 적용 안 함.

### (5) fy > 550MPa 특별 규정
- 횡방향 철근 없음: (c+Ktr)/db ≥ 2.5
- 횡방향 철근 배치: Ktr ≥ 0.25·Atr/s·fyt 및 (c+Ktr)/db ≥ 2.5

## 3. 압축 이형철근 정착 ldc (4.1.3)

**ldc ≥ 200mm**.

기본정착길이 (식 4.1-3):
```
lhb = max(0.25·db·fy / (λ·√fck), 0.043·db·fy)
```

보정계수:
- 소요철근량 초과 배치: `(As,req / As,prov)`
- 나선철근(φ ≥ 6mm, 간격 ≤ 100mm) 또는 D13 띠철근(간격 ≤ 100mm, KDS 14 20 50(4.4.2(3))) 로 둘러싸인 경우: **0.75**

## 4. 다발철근 정착 (4.1.4)

- 다발 내 개별 철근의 ld는 단독 정착길이에 **3개 다발 +20%**, **4개 다발 +33%** 증가.
- 순간격·피복두께·도막 계수 계산 시 다발 전체를 하나의 등가철근으로 취급.

## 5. 표준갈고리 정착 ldh (4.1.5)

**ldh ≥ 8·db 또한 150mm 이상**.

기본정착길이 (식 4.1-4):
```
lhb = (0.24·βr·fy) / (λ·√fck) · db
```
- βr: 에폭시/아연-에폭시 도막 **1.2**, 아연도금 또는 무도막 **1.0**

보정계수:
- D35 이하 철근에서 **측면 피복 ≥ 70mm**, 90°갈고리는 갈고리 넘어선 부분 피복 ≥ 50mm: **0.7**
- D35 이하 90°갈고리에서 ldh 구간을 **3·db 이하 간격**의 띠철근/스터럽이 감싸는 경우: **0.8**
- D35 이하 180°갈고리에서 ldh 구간을 **3·db 이하 간격**의 띠철근/스터럽이 감싸는 경우: **0.8**
- 소요량 초과 배치: `(As,req/As,prov)`

### 불연속단에서 (4.1.5(5))
- 갈고리 양 측면·상하 피복 < 70mm일 때는 **전 ldh 구간을 3·db 이하 간격으로 띠철근/스터럽이 감싸야** 하며, 첫 띠는 갈고리 바깥면 2·db 이내.
- 이 경우 (3)의 0.8 보정은 **적용 불가**.

### fy > 550MPa
- (3)의 0.8 보정 적용 **불가**.

## 6. 확대머리 이형철근 정착 (4.1.6)

**ldt ≥ 8·db, 150mm 이상**.
- 확대머리 **순지압면적 Abrg ≥ 4·Ab**
- **경량콘크리트 불가**, 보통중량콘크리트에만 사용.

접합부 정착 시 (식 4.1-5) 또는 기타 (식 4.1-7).

핵심 조건 (식 4.1-5):
- 순피복두께 ≥ **1.35·db**
- 철근 순간격 ≥ **2·db**
- 확대머리 뒷면이 횡보강철근 바깥면부터 **50mm 이내**.

## 7. 휨부재의 철근 정착 (4.4) — 보 설계 핵심

### 4.4.1 일반 원칙

- 최대응력점과 인장철근이 끝나거나 굽혀진 위험단면에서 정착 검토.
- 휨철근은 더 이상 요구되지 않는 점에서 **max(d, 12·db)** 이상 연장.
  - 단순경간 받침부, 캔틸레버 자유단 제외.
- 연속철근은 단절·굽힘 위치에서 **ld 이상의 묻힘길이** 확보.
- **한 단면에서 인장철근 절단은 전체의 50%를 초과할 수 없음**.
- 인장구역 철근 절단 허용 조건 (아래 중 하나):
  - ① 절단점에서 Vu ≤ (2/3)·φVn
  - ② 절단점에서 **0.75·d 이상 구간에 추가 스터럽**을 필요량 초과로 배치 (Av 초과 ≥ 0.4·bw·s/fyt, s ≤ d/(8·βb))
  - ③ D35 이하이며 연속철근이 필요량의 2배 이상 배치되고 Vu ≤ 0.75·φVn.

### 4.4.2 정모멘트 철근 정착

- **단순부재**: 정모멘트 철근의 **1/3 이상**을 받침부까지 연장. **보는 받침부 내로 150mm 이상** 연장.
- **연속부재**: 정모멘트 철근의 **1/4 이상** 받침부까지 연장.
- 횡하중 저항 주구조물인 경우: 받침부 안쪽 면에서 **fy 발휘 정착**.
- 단순받침부·변곡점에서 정모멘트 철근 (식 4.4-1):
  - `ld ≤ (Mn/Vu) + la`, la는 받침부 내 정착 가능 구간.
  - 반력이 철근 끝을 압축해 구속하면 **30% 증가 가능**.
- 단순받침부에서 표준갈고리 또는 기계적 정착으로 정착 시 식 4.4-1 불필요, 직선 정착 시 식 (4.4-2).
- 깊은보: 받침부 전면에서 **fy 발휘**, 내부 받침부는 인접경간과 연속 또는 겹침이음.

### 4.4.3 부모멘트 철근 정착

- 연속·구속 부재 또는 강결 골조 부재의 부모멘트 철근은 **묻힘·갈고리·기계적 정착**으로 받침부 내 또는 받침부 지나 정착.
- 전체 부모멘트 철근 중 **1/3 이상**은 변곡점을 지나 **max(d, 12·db, ln/16)** 이상 묻힘.
- 깊은보: 내부 받침부에서 인접경간과 연속 설계.

### 4.4.4 복부철근(스터럽) 정착

- 피복·간격 허용 범위 내에서 **부재의 압축·인장면까지 최대한 연장**.
- U형 스터럽 단부 정착 (선택):
  - ① D16 이하 또는 φ16 이하 철선: **종방향 철근을 감싸는 표준갈고리**.
  - ② fy ≥ 300MPa D19/D22/D25: 표준갈고리 + 부재 중간깊이 (h/2)에서 갈고리 단부 밖까지 **0.17·d·fy/√fck 이상 묻힘**.
  - ③ 용접철망: 세부 가닥 정착 규정 (두 개의 종방향 철선 50mm 이상 간격).
- 장선구조(KDS 14 20 10(4.11))에서 D13 이하/13mm 이하 스터럽: **표준갈고리** 필수.
- **폐쇄형 U형 스터럽/띠철근**은 **겹침이음길이 1.3·ld 이상**일 때 적절한 이음.
  - 깊이 ≥ 450mm이고 스터럽 가닥이 전 깊이까지 연장되면 폐쇄스터럽 이음 적절.
  - 한 가닥 이음부 인장력 Ab·fy ≤ **40 kN**.

## 8. 철근 이음 (4.5)

### 4.5.1 이음 일반

- **D35 초과 철근은 겹침이음 불가** (4.5.3(2) 예외).
- 다발철근 겹침이음은 개별철근 기준 ld에 4.1.4 증가 적용, 다발 내 각 철근 이음 위치는 엇갈리게.
- 휨부재에서 비접촉 겹침이음 철근은 횡방향으로 **max(ls/5, 150mm)** 이상 떨어지지 말아야 함.
- **용접이음·기계적이음은 fy의 125% 이상** 강도 발휘.
- 굽힘부에서 용접이음 불가, 굽힘 시작부터 **2·db 이상** 떨어진 곳부터 가능.

### 4.5.2 인장 이형철근/이형철선 이음

겹침이음길이 ≥ **300mm**.

| 등급 | 길이 | 적용조건 |
| ---- | ---- | -------- |
| A급 | **1.0·ld** | 배치 철근량 ≥ 소요량×2 AND 이음부에서 겹침이음된 철근량 ≤ 전체의 1/2 |
| B급 | **1.3·ld** | A급에 해당 안 되는 모든 경우 |

- 크기가 다른 철근 인장 겹침: 큰 철근의 ld와 작은 철근의 겹침이음길이 중 **큰 값**.
- 기계적이음 등급(KCI-ST103) 선택은 배치철근량·이음위치 비율로 구분.

### 4.5.3 압축 이형철근 이음

겹침이음길이 (식 4.5-1, 4.5-2):
- `ls = 0.072·fy·db` (fy ≤ 400MPa)
- `ls = (0.13·fy − 24)·db` (fy > 400MPa)
- **≥ 300mm**, fck < 21MPa이면 **1/3 증가**.
- 4.5.2 인장 이음길이보다 길 필요 없음.
- D41, D51 철근은 D35 이하와 겹침이음 가능.

### 단부 지압이음 (4.5.3(4),(5))
- 철근 양 단부를 직각 절단 후 밀착 접촉.
- 오차: 절단면 1.5° 이내, 조립 후 지압면 3° 이내.
- **폐쇄띠철근·폐쇄스터럽·나선철근이 있는 압축부재에서만** 사용.

## 9. 보 정착·이음 설계 체크리스트

### 정착
- [ ] 인장 이형철근 ld 계산 (식 4.1-1 또는 4.1-2) ≥ 300mm
- [ ] 상부철근이면 α = 1.3 반영
- [ ] 표준갈고리 정착 시 ldh ≥ 8·db, 150mm
- [ ] 확대머리 정착 시 순지압면적 ≥ 4·Ab

### 휨철근 배치
- [ ] 휨철근 절단 위치에서 **d 또는 12·db** 이상 연장
- [ ] 한 단면에서 철근 50% 초과 절단 금지
- [ ] 단순보: 정모멘트 철근 **1/3 이상** 받침부 연장 (150mm 이상)
- [ ] 연속보: 정모멘트 철근 **1/4 이상** 연장, 부모멘트 철근 **1/3 이상** 변곡점 지나 max(d, 12db, ln/16) 연장

### 이음
- [ ] D35 초과 겹침이음 배제
- [ ] 인장 겹침이음: A급(1.0·ld) 조건 확인, 아니면 B급(1.3·ld)
- [ ] 압축 겹침이음 ls 계산, ≥ 300mm
- [ ] 용접·기계적 이음: **1.25·fy 이상** 발휘 능력
- [ ] 스터럽 이음: 1.3·ld 또는 h ≥ 450mm에서 전 깊이 연장
