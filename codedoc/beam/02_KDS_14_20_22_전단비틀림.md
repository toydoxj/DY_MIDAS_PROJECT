# KDS 14 20 22 콘크리트구조 전단 및 비틀림 설계기준 (2022-01-11)

> RC 보의 전단·비틀림 설계 절차와 제한사항을 발췌 정리. 원문 `../raw/KDS_14_20_22_raw.txt`.

## 1. 전단 설계 원칙 (4.1)

기본식: **φ·Vn ≥ Vu** (식 4.1-1)
- Vn = Vc + Vs (식 4.1-2)
- Vc: 콘크리트에 의한 전단강도 (4.2)
- Vs: 전단철근에 의한 전단강도 (4.3)

### 제한

- **√fck**의 값은 **8.4 MPa**를 초과하지 않는다.
  - 단, 최소 전단철근이 배치된 RC/PSC 보 및 장선에서는 Vc, Vci, Vcw 계산 시 예외.
- 위험단면 위치 (4.1.1(5)):
  - 받침부 반력이 부재 단부를 압축하고, 하중이 윗면에 작용하며, 받침부~위험단면 사이에 집중하중이 없을 때:
    - RC: 받침부 내면에서 **d** 거리의 단면
    - PSC: 받침부 내면에서 **h/2** 거리의 단면
- 깊은보, 브래킷, 벽체, 슬래브/기초판은 4.7–4.11의 특별 규정 추가 적용.

## 2. 콘크리트에 의한 전단강도 Vc (4.2)

### 2.1 철근콘크리트 (4.2.1)

일반식(상세계산 안 할 때):
- 전단+휨만: `Vc = (1/6)·λ·√fck·bw·d` (식 4.2-1, 기호 단순화)
- 축방향 **압축력** 작용: 식 (4.2-2)
- 큰 축방향 **인장력**: 전단철근으로 전 전단력 저항 (식 4.2-6 상세 사용하지 않을 시)

정밀식 (상세계산 가능):
- 식 (4.2-3): `Vc = (0.16·λ·√fck + 17·ρw·Vu·d/Mu)·bw·d ≤ 0.29·λ·√fck·bw·d`
  - `Vu·d/Mu ≤ 1.0`
- 축 압축/인장 시 식 (4.2-4), (4.2-5), (4.2-6).

원형단면: 단면적 계산 시 **d = 0.8·Dia**로 가능.

### 2.2 프리스트레스트콘크리트 (4.2.2)

- 유효프리스트레스 ≥ 0.4·f_pu일 때 식 (4.2-7) 약식 적용.
- 상세식: Vci, Vcw 중 작은 값 (식 4.2-8 ~ 4.2-10).
- 프리텐션 전달길이 내에서는 감소된 프리스트레스 힘 고려 (4.2.2(4)).

## 3. 전단철근 Vs (4.3)

### 3.1 전단철근 형태 (4.3.1)

사용 가능한 형태:
- 부재축에 직각인 **스터럽**
- 부재축에 직각인 **용접철망**
- 나선철근, 원형 띠철근, 후프철근

RC 부재에서 추가 허용:
- 45° 이상 경사 스터럽
- 30° 이상 굽힘철근
- 스터럽 + 굽힘철근 조합

제한:
- **fyt ≤ 500 MPa** (벽체 또는 용접 이형철망은 600 MPa).
- PSC에서 d = 압축연단~(긴장재·철근 도심) 거리, **d ≥ 0.8·h**.
- 스터럽은 KDS 14 20 52(4.4.4)에 따라 정착.

### 3.2 스터럽 간격 제한 (4.3.2)

| 조건 | 최대 간격 smax |
| ---- | ------------- |
| RC, 일반 | **d/2**, 600mm 이하 |
| PSC, 일반 | **0.75·h**, 600mm 이하 |
| Vs > (1/3)·√fck·bw·d | 위 값의 **절반** |
| 경사스터럽·굽힘철근 | 부재 중간 높이에서 반력점 방향 45°선과 1회 이상 교차 |

### 3.3 최소 전단철근 (4.3.3)

적용: `Vu > 0.5·φVc`인 모든 RC/PSC 휨부재.

**예외** (최소 전단철근 불필요):
- 슬래브 및 기초판
- 콘크리트 장선구조 (KDS 14 20 10(4.3.11))
- 전체 깊이 ≤ 250mm인 보, 또는 I/T형 보에서 h ≤ max(2.5·t_f, bw/2)
- 교대 벽체·날개벽·옹벽·암거 등 휨이 주거동인 판부재
- 속빈 부재로 h ≤ 315mm이며 Vu ≤ Vc_한계
- 강섬유 콘크리트 보(h ≤ 600mm, fck ≤ 40MPa)에서 Vu가 한계치 이하

**최소 전단철근량** (식 4.3-1):
- `Av,min = 0.0625·√fck·(bw·s/fyt) ≥ 0.35·bw·s/fyt`

### 3.4 전단철근 설계 (4.3.4)

- 직각 스터럽: **`Vs = Av·fyt·d/s`** (식 4.3-3).
- 원형 띠/후프/나선: 식 4.3-3 사용하되 Av는 s 내 두 가닥 단면적.
- 경사스터럽: `Vs = Av·fyt·(sinα+cosα)·d/s` (식 4.3-4).
- 굽힘철근 1개: `Vs = Av·fyt·sinα ≤ 0.25·√fck·bw·d` (식 4.3-5).
- **`Vs ≤ (2/3)·√fck·bw·d` 상한** (4.3.4(9)).

## 4. 비틀림 설계 (4.4 ~ 4.5)

### 4.1 비틀림 무시 조건 (4.4.1)

계수비틀림모멘트 Tu가 다음 임계값 이하면 비틀림 무시 가능:
- RC 부재: `Tu < φ·(√fck/12)·(Acp²/pcp)` (식 4.4-1)
- PSC 부재: 식 (4.4-2) (프리스트레스 영향 고려)
- 축인장/압축 받는 RC: 식 (4.4-3)
- 속빈 단면: Ag를 Acp 대신 사용 가능.

### 4.2 계수비틀림모멘트 계산 (4.4.2)

- **적합 비틀림**(부정정 구조에서 균열 후 재분배 가능): Tu를 식 (4.4-4)~(4.4-6)으로 감소 가능.
- 받침부 내 h/2 이내(RC) / h/2 이내(PSC) 단면은 **h/2에서 계산된 Tu 이상**으로 설계.

### 4.3 비틀림강도 (4.5.1)

단면치수 검증:
- 속찬 단면 (식 4.5-1): `√((Vu/(bw·d))² + (Tu·ph/(1.7·Aoh²))²) ≤ φ·(Vc/(bw·d) + (2/3)·√fck)`
- 속빈 단면 (식 4.5-2): `Vu/(bw·d) + Tu·ph/(1.7·Aoh²) ≤ 위 우변`
- **비틀림철근 fyt ≤ 500 MPa**.

### 4.4 비틀림철근 산정 (4.5.2)

- Tn 계산 시 Vc는 비틀림 영향 받지 않는다고 가정.
- 수직철근 (폐쇄스터럽): 식 (4.5-4). 압축경사각 θ:
  - 프리스트레스 없음 또는 프리스트레스 < 0.4·fpu → **θ = 45°**
  - 프리스트레스 ≥ 0.4·fpu → **θ = 37.5°**
- 추가 종방향 철근 Al (식 4.5-5): 폐쇄스터럽 둘레 따라 분포.

### 4.5 비틀림철근 상세 (4.5.3)

- 폐쇄스터럽/폐쇄띠철근/폐쇄용접철망 또는 나선철근.
- 횡방향 비틀림철근 정착: **135° 표준갈고리로 종방향 철근 감싸기** 또는 KDS 14 20 52(4.4.4(2)①,②,③).
- 종방향 비틀림철근은 **양단에 정착**.
- 속빈 단면: 횡방향 비틀림철근 중심선~내부 벽면 거리 ≥ **0.5·Aoh/ph**.

### 4.6 최소 비틀림철근 및 간격 (4.5.4)

- 최소 폐쇄스터럽 단면적 (식 4.5-6): `(Av+2At) ≥ 0.0625·√fck·(bw·s/fyt) ≥ 0.35·bw·s/fyt`
- 최소 종방향 비틀림철근 (식 4.5-7): `Al,min = (5·√fck·Acp/(12·fyt)) − (At/s)·ph·(fyt/fyl)`, 단 At/s ≥ 0.175·bw/fyt.
- **횡방향 비틀림철근 간격 ≤ ph/8, 또한 300mm 이하**.
- **종방향 비틀림철근**은 폐쇄스터럽 둘레 따라 **300mm 이하** 간격, 스터럽 모서리마다 최소 1개, **dl ≥ max(s/24, D10)**.
- 비틀림철근은 계산상 필요한 위치에서 **(bt+d) 이상** 거리까지 연장.

## 5. 깊은보의 전단 설계 (4.7)

적용: ln/h ≤ 4 또는 하중점~받침부 거리가 부재 깊이의 2배 이내이고 하중·받침부가 반대면이어서 압축대 형성.

- 설계: KDS 14 20 20(4.2.4) 비선형해석 또는 KDS 14 20 24 스트럿-타이모델.
- **Vn ≤ (5/6)·√fck·bw·d 한계** (4.7.1(3)).
- 최소 수직전단철근 Av ≥ 0.0025·bw·s, 간격 s ≤ min(d/5, 300mm).
- 최소 수평전단철근 Avh ≥ 0.0015·bw·s2, 간격 s2 ≤ min(d/5, 300mm).
- 또는 KDS 14 20 24(4.2.3) 요구 조건 만족.

## 6. 보 전단·비틀림 설계 체크리스트

### 입력
- [ ] 단면 bw, h, d, Aoh, ph, Acp, pcp
- [ ] fck, fyt
- [ ] 해석 결과 Vu, Tu, Pu

### 전단
- [ ] 위험단면(받침부에서 d 또는 h/2)의 Vu 사용
- [ ] Vc 계산 (식 4.2-1 또는 4.2-3)
- [ ] Vu > φ·Vc이면 스터럽 배치, `Vs = (Vu/φ − Vc)`
- [ ] **Vs ≤ (2/3)·√fck·bw·d**
- [ ] 간격: d/2 이하(또는 (1/3)√fck 초과 시 d/4), 600mm 이하
- [ ] Vu > 0.5·φVc인 구간은 최소 스터럽 확인 (식 4.3-1)

### 비틀림
- [ ] Tu가 임계값(식 4.4-1 등)을 초과하는지 확인
- [ ] 단면 치수 검증 (식 4.5-1/2)
- [ ] 폐쇄스터럽 및 종방향 철근 계산
- [ ] 간격: 스터럽 ph/8 또는 300mm, 종방향 300mm

### 깊은보
- [ ] ln/h ≤ 4 또는 2h 이내 집중하중 → 4.7 적용
- [ ] Vn ≤ (5/6)·√fck·bw·d 제한
- [ ] 수직·수평 전단철근 최소량 확인
