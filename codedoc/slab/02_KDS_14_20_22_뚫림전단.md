# KDS 14 20 22 콘크리트구조 전단 및 비틀림 설계기준 — 2방향 뚫림전단 (2022-01-11)

> 슬래브·기초판의 2방향 거동에서 기둥 주변 뚫림전단(punching shear) 설계를 발췌 정리. 원문 `../raw/KDS_14_20_22_raw.txt` (4.11 섹션).

## 1. 전단 설계 단면 (4.11.1)

집중하중이나 반력 부근에서 슬래브·기초판은 다음 **두 경우 중 불리한 경우**로 검토:

- **(2) 1방향 거동(폭이 넓은 보처럼 휨)**: 전체 폭 단면, **4.1 ~ 4.3** 규정(보의 전단설계) 적용.
- **(3) 2방향 거동**: **4.11.2 ~ 4.11.7** 규정 적용.
  - **위험단면 둘레 bo** 는 **최소로 되도록** 하되 집중하중·반력구역·기둥·기둥머리·지판 경계로부터 **d/2 보다 가까이 둘 필요 없음**.
  - 사각형 기둥 위험단면은 네 변에 나란한 직선으로 정의 가능.

## 2. 2방향 거동 공칭전단강도 Vc (4.11.2)

### 2.1 기본식 — RC 슬래브·기초판
설계: `φVn ≥ Vu`, Vn = Vc + Vs.

```
Vc = vc · bo · d             (식 4.11-1)

vc = λ·ks·kbo · f_te · cot(θ) · c_u  
    + η · f_cc               (식 4.11-2)
```

여기서:
- **vc**: 콘크리트 공칭전단응력강도
- **bo**: 위험단면 둘레길이 (기둥면에서 d/2)
- **λ**: 경량콘크리트계수
- **ks**: 슬래브 두께계수, 식 (4.11-3): **`ks = (300/d)^0.25 ≥ 0.75`** (d: mm)
- **kbo**: 위험단면 둘레길이 영향계수 (내부 기둥 1.0, 외부 1.33, 모서리 2.0) — 식 (4.11-4)
- **f_te**: 압축대 콘크리트 인장강도 — 식 (4.11-5)
- **θ**: 슬래브 휨 압축대 균열각도
- **cu**: 압축대 깊이 평균값 — 식 (4.11-6)
- **fcc**: 위험단면 압축대 평균 압축응력 — 식 (4.11-7), (4.11-8)

**주인장철근비 ρ 적용 범위 (식 4.11-7)**:
- 기둥면에서 양방향으로 각각 일정 거리 확대된 폭의 **평균 주인장철근비**.
- **ρ ≤ 0.005** 이면 ρ = 0.005 사용.
- 주인장철근은 위험단면에서 최소 **(bo 영역 + ld)** 이상 슬래브에 정착 필요.

### 2.2 PSC 슬래브·기초판 (4.11.2(3))
```
Vc = (β_p · λ · √fck + 0.3 · fpc) · bo · d + Vp     (식 4.11-9)
```
- **Vp**: 위험단면에서 유효프리스트레스 힘의 **수직성분**.
- **fpc**: 위험단면의 평균 콘크리트 압축응력.

식 (4.11-9) 적용 조건 (4.11.2(4)):
- ① 기둥단면의 어느 부분도 불연속단에서 **4·t 이내**에 있지 않아야 함.
- ② **fck ≤ 35 MPa**.
- ③ 각 방향 프리스트레스 압축응력 **0.88 ≤ fpc ≤ 3.5 MPa**.

조건 미만족 시 → **식 (4.11-1)** 적용.

### 2.3 Vn 상한 (4.11.2(5))
- **Vn ≤ (2/3)·√fck · bo · d** 범위로 제한.

### 2.4 Vu 산정 시 무시 가능 영역 (4.11.2(6))
- **슬래브-기둥 접합부**: 기둥면에서 **0.5·d** 내 등분포하중 무시 가능.
- **기초판-기둥 접합부**: 기둥면에서 **0.75·d** 내 등분포 지반력 무시 가능.

## 3. 뚫림전단 전단철근 (4.11.3)

### 적용 범위
- 슬래브 두께 **d ≥ 150 mm** 이고 **전단철근 지름의 16배 이상**일 때 적용.

### Vs 계산 (식 4.11-10)
```
Vs = (Av · fyt) / s
```
- Av: 위험단면 둘레 따라 배치된 전단철근 **총 단면적**.
- **fyt ≤ 400 MPa**.

### 한계
- **Vn ≤ (5/6)·√fck · bo · d** 범위.

### 스터럽 간격 제한

| 위치 | 최대 간격 |
| ---- | --------- |
| 기둥면 → 첫 번째 열 스터럽 | **d/2 이하** |
| 첫 번째 열 내 기둥면 평행 스터럽 다리 간격 | **2·d 이하** |
| 스터럽 열 사이 (기둥면 직각방향) | **d/2 이하** |

### 정착
- 슬래브 전단철근은 **상·하 단면에서 충분히 정착** (KDS 14 20 52(4.4) 만족).
- 스터럽은 **길이방향 휨철근을 둘러싸야** 함.

## 4. 전단머리(I형강·ㄷ형강) 보강 (4.11.4)

| 규정 | 내용 |
| ---- | ---- |
| (2) | I 또는 ㄷ형강을 **완전 용입 용접**으로 조립, 기둥단면 내에서 중단 금지 |
| (3) | **전단머리 높이 ≤ 70·tw** (강재 복부 두께의 70배) |
| (4) | 단부 절단은 수평과 **30° 이상** 각, 가늘어진 단면 소성모멘트 충분 |
| (5) | 강재 **압축플랜지**는 슬래브 압축면에서 **0.3·d 이내**에 위치 |
| (6) | 강재 강성 / 인접 슬래브 폭 (c1+d) 합성균열 단면 강성 **비 αv ≥ 0.15** |
| (7) | 각 전단머리 소요 소성휨모멘트 Mp = (식 4.11-11) |
| (8) | 위험단면 위치: 기둥면에서 lv−c1/2 거리의 **3/4** 위치에서 각 전단머리와 교차 |
| (9) | **Vn ≤ (1/3)·√fck · bo · d** (위험단면), 전단머리 사용 시 **Vn ≤ (7/12)·√fck · bo · d** (4.11.1(3) 위험단면) |
| (10) | 전단머리가 슬래브 주열대 휨모멘트에 기여하는 양 = (식 4.11-12), 다만 **주열대 전체 계수휨모멘트의 30% 이하** 등 |
| (11) | 불균형모멘트 Mv 고려 시 전단머리 적절히 정착 |

## 5. 확대머리 전단스터드(Headed Shear Studs) (4.11.5)

- 슬래브/기초판에 **수직**으로 배치. 부속철물 포함 스터드 총 높이 ≥ (부재 두께 − 상하 피복 − 주철근 지름의 절반).
- Vn: 4.11.3(2),(3) 식 사용, Av = 기둥면 둘레방향 한 열 내 전단스터드 **총 단면적**.
- 기둥면 → 첫 번째 열 간격 **≤ d/2**.
- 열 사이 간격(기둥면 직각방향)은 **일정**, PSC 조건 만족 시 **≤ 0.75·d**, 일반 RC **≤ 0.5·d**.
- 위험단면 모서리 양단에 **최소 1개씩** 스터드, 모서리 간 간격 **≤ 2·d**.

## 6. 슬래브 개구부 (4.11.6)

- 집중하중·반력 면에서 **슬래브 두께 10배 이내** 개구부, 또는 플랫 슬래브 주열대 내 개구부일 때 **위험단면 수정**:
  - (2) 전단머리 없음: 기둥/하중 중심 ↔ 개구부 경계 직선 내 포함되는 위험단면 둘레는 **유효하지 않음**.
  - (3) 전단머리 있음: 위 무효 둘레의 **절반**만 무효 처리.

## 7. 휨모멘트 전달 편심전단 (4.11.7)

슬래브·기초판과 기둥 사이에 **불균형휨모멘트 Mu**가 발생하면 접합부는 이를 전달하도록 설계.

### 기본 설계식 (식 4.11-13)
```
Mu ≤ φMn_v
```
- φMn_v: 강도감소계수가 고려된 위험단면의 설계불균형휨강도.

### 3성분 합으로 계산 (식 4.11-14)
```
Mn_v = Mf + M_ef + M_es
```
- **Mf**: 전면+후면 슬래브 **휨강도 합** (식 4.11-15) — γf 분담.
- **Mef**: 전면·후면의 **편심전단에 의한 휨강도**.
- **Mes**: 측면의 **편심전단에 의한 비틀림강도**.
- γf + γv = 1 (휨 분담 + 편심전단 분담 = 1).
- 위험단면 4면 구분: **전면**(직접전단과 불균형모멘트의 전단 방향이 일치하는 면) / **후면** / **측면**(두 방향 직교면).

### Mf — 슬래브 휨분담 (식 4.11-15)
- 전면 상부 주철근과 후면 하부 주철근을 인장철근으로 하는 슬래브 휨강도.
- 휨인장철근은 위험단면 폭에서 양방향으로 **최소 일정 거리 확대**하여 배치, **ρ ≤ 0.03** 제한.
- 외부접합부에서는 Mf_front 또는 Mf_rear 중 하나가 존재하지 않을 수 있음.

### Mef, Mes — 편심전단 분담
- 위험단면 **각 모서리의 전단력 × 기둥 중심까지 거리**.
- 위험단면에서 중립축부터 **선형 전단응력 분포** 가정, 힘 평형으로 중립축 위치 결정.

### 각 면 편심전단강도

**전면·후면** (식 4.11-16):
```
V_ef ≤ (v_c + f_p) · A_ef + V_s,ef
```
- v_c: 식 (4.11-2) 기준.
- f_p: 전면·후면 유효프리스트레스 수직성분에 의한 위험단면적 평균응력.
- V_s,ef: 전면·후면 전단보강재에 의한 편심전단강도.
  - Vs(식 4.11-10)를 전·후·측면에 **균등 배치** 조건으로 산정, **fyt ≤ 400 MPa**.

**측면** (식 4.11-17):
```
V_es ≤ (v_c + f_p,s) · A_es + V_s,es
```

### 양방향 불균형모멘트 상호작용 (4.11.7(5))
```
(Mu,1 / φMn,1)^α + (Mu,2 / φMn,2)^α ≤ 1.0   (식 4.11-18)
```

### 전단머리 사용 시 (4.11.7(6))
- 4.11.4(8) 위험단면에 작용하는 연직하중 전단응력 + 전단편심에 의한 휨모멘트 전단응력의 합 **≤ (1/3)·√fck** 초과 금지.

## 8. 뚫림전단 설계 체크리스트

### 입력
- [ ] 기둥 치수 c1 × c2, 슬래브 두께 t, 유효깊이 d
- [ ] fck, fyt (전단철근), 경량콘크리트 여부
- [ ] 계수 연직하중 Vu, 불균형모멘트 Mu

### 위험단면
- [ ] bo = 기둥면에서 d/2 둘레
- [ ] 개구부 영향(4.11.6) 반영, 위험단면 무효 둘레 제외
- [ ] 내부/외부/모서리 기둥 kbo (1.0/1.33/2.0)

### 강도 검증
- [ ] Vc (식 4.11-1 또는 4.11-9) 계산
- [ ] 주인장철근비 ρ 0.005 이상/0.03 이하 확인, 정착길이 연장
- [ ] Vn ≤ (2/3)·√fck·bo·d (4.11.2(5))
- [ ] Vu ≤ φVn 확인 (등분포 무시영역 0.5d / 0.75d 반영)

### 전단보강 필요 시
- [ ] d ≥ 150mm, dbar ≤ d/16
- [ ] Vs = Av·fyt/s, fyt ≤ 400 MPa
- [ ] Vn ≤ (5/6)·√fck·bo·d
- [ ] 스터럽 간격: 첫 열 d/2, 열 간 d/2, 기둥면 평행 다리 2d
- [ ] 또는 전단머리 4.11.4 / 확대머리 스터드 4.11.5 옵션 적용

### 불균형모멘트 (4.11.7)
- [ ] γf·Mu는 슬래브 휨으로 (0.5d 폭 내), γv·Mu는 편심전단으로
- [ ] 양방향이면 상호작용식 (식 4.11-18) 만족
- [ ] 식 4.11-13 `Mu ≤ φMn_v` 확인
