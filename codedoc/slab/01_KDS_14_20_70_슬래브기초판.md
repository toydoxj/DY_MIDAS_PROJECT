# KDS 14 20 70 콘크리트 슬래브와 기초판 설계기준 (2021-02-18)

> 슬래브(1방향·2방향·플랫)와 기초판의 해석·설계 절차, 배근 상세를 발췌 정리. 원문 `../raw/KDS_14_20_70_raw.txt`.

## 1. 슬래브 설계 (4.1)

### 1.1 1방향 슬래브 (4.1.1)

#### 설계 원칙 (4.1.1.1)
- **마주보는 두 변에만** 지지되는 슬래브는 KDS 14 20 20에 따라 설계.
- 4변 지지 2방향 슬래브 중 **장변/단변 > 2** 이면 **1방향으로 해석**하며, 단변방향 경간 사용.
- 경간 정의: **KDS 14 20 10(4.7)** 참조.

#### 보와 일체 연속 슬래브 해석 (4.1.1.2)
단순받침부 위 연속보로 가정한 **탄성해석** 또는 KDS 14 20 10(4.1) 근사법 사용. 보정:
- 활하중 경간 중앙 **부모멘트**는 산정값의 **1/2**만 취함.
- 경간 중앙 **정모멘트**는 양단 고정으로 계산한 값 이상.
- 순경간 > 3.0 m면 순경간 내면 휨모멘트 사용 가능(단, 순경간 기준 고정단 휨모멘트 이상).
- 양단부 보의 처짐차 영향 고려.

#### 구조 상세 (4.1.1.3)
| 항목 | 규정 |
| ---- | ---- |
| **슬래브 두께** | KDS 14 20 30(4.2.1)에 따르며 **≥ 100 mm** |
| **위험단면 철근 중심 간격** | **≤ min(2t, 300 mm)** |
| **기타 단면 철근 간격** | **≤ min(3t, 450 mm)** |
| **수축·온도철근** | 주모멘트 철근 **직각방향**으로 KDS 14 20 50(4.6.2) 배치 |
| **내민슬래브 부모멘트** | 슬래브 끝 단순받침부에서도 부모멘트 철근 배치 |
| **장변방향 상부 균열철근** | 단변방향 보 상부의 부모멘트 균열 방지, KDS 14 20 10(4.10(3)) |

## 2. 2방향 슬래브 설계 절차 (4.1.2)

### 2.1 정의 (4.1.2.1)

- **유효지지단면**: 기둥축 중심으로 **45°** 내로 펼쳐진 기둥·기둥머리·브래킷 내에 위치하는 최대 정원추/정사면추/쐐기 모양 표면과 슬래브(또는 지판) 바닥의 절단면.
- **주열대(column strip)**: 기둥 중심선 양쪽으로 **min(0.25·l1, 0.25·l2)** 씩 폭. 받침부 사이 보 포함.
- **중간대(middle strip)**: 두 주열대 사이 영역.
- **보의 유효 단면(slab 일체)**: 보가 슬래브 위/아래로 내민 깊이 중 큰 값을 보 양측으로 연장, **한쪽 연장 ≤ 4·t**.
- **전단머리**: 슬래브-기둥 접합부에서 위험단면 확장을 위해 슬래브 아래로 돌출, 돌출 두께만큼 기둥면에서 최소 위험단면 확장.

### 2.2 해석 및 설계 방법 (4.1.2.2)

- 평형조건 + 기하학적 적합조건 만족하는 어떤 방법이든 가능 (FEM 포함).
- 연직하중에 대해 **직접설계법(4.1.3)** 또는 **등가골조법(4.1.4)** 사용 가능.
- 횡방향 변위가 발생하는 골조에서는 철근·균열 영향 고려 부재 강성.
- 슬래브 시스템이 횡하중을 받으면 횡력+연직하중 조합 해석.

### 2.3 불균형휨모멘트 전달 (4.1.2.3)
- 연직·풍·지진하중 등으로 슬래브-기둥 간 모멘트 전달 시 **KDS 14 20 22(4.11.7)** 에 따라 설계.
- 휨에 의해 전달되는 불균형모멘트 분담분 γf·Mu는 **기둥(기둥머리) 면에서 양쪽 0.5·t 폭** 슬래브 위험단면 내에서 전달된다고 간주.

### 2.4 플랫 슬래브의 지판(drop panel) (4.1.2.4)
- 받침부 중심선에서 각 방향 **경간의 1/6 이상** 연장.
- 돌출 두께 **≥ 슬래브 두께의 1/4**.
- 철근량 계산 시 돌출 두께는 **지판 외단부~기둥(머리) 면 거리의 1/4 이하**로 취급.

## 3. 직접설계법 (4.1.3)

### 3.1 적용 제한 (4.1.3.1)
다음 **모두 만족** 시에만 직접설계법 사용 가능.

| # | 제한사항 |
| -- | -------- |
| (2) | 각 방향 **3경간 이상 연속** |
| (3) | 슬래브 판은 **장단변비 ≤ 2** 직사각형 |
| (4) | 연속 받침부 중심간 경간 차이 ≤ **긴 경간의 1/3** |
| (5) | 기둥 중심선 어긋남 ≤ **경간의 10%** |
| (6) | 모든 하중 **등분포 연직하중**, 활하중 ≤ **2·고정하중** |
| (7) | 보 있는 경우 두 방향 보의 상대강성 식 (4.1-1) 만족 |
| (8) | 휨모멘트 재분배(KDS 14 20 10(4.2)) 적용 불가. 단, 4.1.3.7의 10% 수정은 허용 |
| (9) | 제한 다소 벗어나도 4.1.2.2 해석으로 입증 시 적용 가능 |

### 3.2 전체 정적 계수휨모멘트 Mo (4.1.3.2)

```
Mo = wu · l2 · ln² / 8      (식 4.1-2)
```
- **정+부 계수휨모멘트 절댓값 합 ≥ Mo**.
- l2: 직각방향 경간 (양측이 다르면 평균).
- 가장자리 평행 경간: l2 = 가장자리 ~ 슬래브 판 중심선 거리.
- ln: 기둥/기둥머리/브래킷/벽체 내면 사이 순경간, **≥ 0.65·l1**.
- 원형·정다각형 받침부는 **같은 면적의 정사각형**으로 취급.

### 3.3 정 및 부계수휨모멘트 분배 (4.1.3.3)

**내부 경간**:
- 부 = **0.65·Mo**
- 정 = **0.35·Mo**

**단부 경간** (표 4.1-1):

| 구분 | (1) 구속되지 않은 외부 받침부 | (2) 보 모든 받침부 사이 | (3) 내부 받침부 사이 보 없음, 테두리보 없음 | (4) 테두리보 있음 | (5) 완전 구속 외부 받침부 |
| ---- | --- | --- | --- | --- | --- |
| 내부 받침부 부모멘트 | 0.75 | 0.70 | 0.70 | 0.70 | 0.65 |
| 정모멘트 | 0.63 | 0.57 | 0.52 | 0.50 | 0.35 |
| 외부 받침부 부모멘트 | 0 | 0.16 | 0.26 | 0.30 | 0.65 |

- 받침부 불균형모멘트는 인접부재 강성 따라 분배 해석 또는 양측 중 **큰 부모멘트**에 견디도록 설계.
- 슬래브 단부/테두리보는 외부 받침부 부모멘트만큼 **비틀림**에 견디도록 설계.
- 외부 기둥-슬래브 연직하중 모멘트 = **0.3·Mo**.

### 3.4 주열대 계수휨모멘트 분배 (4.1.3.4)

**표 4.1-2 — 주열대 내부 받침부 부모멘트 분배 (%)**

| l2/l1 | 0.5 | 1.0 | 2.0 |
| ----- | --- | --- | --- |
| α1·l2/l1 = 0 | 75 | 75 | 75 |
| α1·l2/l1 ≥ 1.0 | 90 | 75 | 45 |

**표 4.1-3 — 주열대 외부 받침부 부모멘트 분배 (%)**

| l2/l1 | 0.5 | 1.0 | 2.0 |
| ----- | --- | --- | --- |
| α1·l2/l1 = 0 / βt = 0 | 100 | 100 | 100 |
| α1·l2/l1 = 0 / βt ≥ 2.5 | 75 | 75 | 75 |
| α1·l2/l1 ≥ 1.0 / βt = 0 | 100 | 100 | 100 |
| α1·l2/l1 ≥ 1.0 / βt ≥ 2.5 | 90 | 75 | 45 |

**표 4.1-4 — 주열대 정모멘트 분배 (%)**

| l2/l1 | 0.5 | 1.0 | 2.0 |
| ----- | --- | --- | --- |
| α1·l2/l1 = 0 | 60 | 60 | 60 |
| α1·l2/l1 ≥ 1.0 | 90 | 75 | 45 |

> 중간 값은 **직선보간**.

- **l2 ≥ 3/4 의 경간이 기둥·벽체 지지** 시 부모멘트는 l2 따라 **균등 분포**로 간주.
- 보가 있는 슬래브의 주열대는 보가 부담하지 않는 몫에 견디도록 설계.

### 3.5 보의 계수휨모멘트 (4.1.3.5)
- **α1·l2/l1 ≥ 1.0**: 보는 **주열대 휨모멘트의 85%** 부담.
- **0 < α1·l2/l1 < 1.0**: 85% ~ 0% 직선보간.
- 보는 추가로 보 위 내민 자중·집중하중·분포하중 모멘트에 견디도록 설계.

### 3.6 중간대 계수휨모멘트 (4.1.3.6)
- 주열대 미부담 정·부 모멘트를 **주열대 양쪽 1/2 중간대**에 비례 할당.
- 각 중간대 = **2개의 1/2 중간대 합**에 견디도록 설계.
- 벽체 지지 가장자리 인접 중간대는 첫 내부 받침부 1/2 중간대 할당값의 **2배**.

### 3.7 계수휨모멘트 수정 (4.1.3.7)
- 전체 Mo가 식 (4.1-2) 이상이라면 정·부 모멘트를 **±10% 수정 가능**.

### 3.8 보가 있는 슬래브의 전단력 (4.1.3.8)
- α1·l2/l1 ≥ 1인 보: 슬래브 **네 모퉁이 45°선** + 장변 평행 슬래브 중심선으로 둘러싸인 재하면적 하중 부담.
- α1·l2/l1 < 1: 0(α1l2/l1=0)일 때 보가 하중을 받지 않는다고 가정해 직선보간.
- 슬래브는 전 전단력에 견디도록 설계, 전단강도는 **KDS 14 20 22**.

### 3.9 기둥·벽체 계수휨모멘트 (4.1.3.9)
- 기둥·벽체는 슬래브 하중으로 발생하는 모멘트에 견디도록.
- 내부 받침부에서 상하 부재 강성에 비례해 (식 4.1-3) 부담.

## 4. 등가골조법 (4.1.4)

### 4.1.4.1~4.1.4.7 요약

- **등가골조(4.1.4.2)**: 건물 전체를 가로·세로 기둥선 따라 2D 골조로 분리 해석.
- **슬래브(4.1.4.3)**: 경간 내 단면 변화 고려한 슬래브+보 단면 강성 사용.
- **기둥(4.1.4.4)**: 접합부/기둥머리 바깥 I는 전 단면 기준. 기둥 강성은 **비틀림 부재 강성과 조합**.
- **비틀림 부재(4.1.4.5)**: 기둥과 직교하는 슬래브+보 일부를 비틀림 부재로 모델링.
- **활하중 배치(4.1.4.6)**: 알려진 경우 그대로, 미상이면 **각 경간에 활하중 3/4** 동시작용 + 인접경간만 활하중 케이스.
- **계수휨모멘트(4.1.4.7)**: 내부 받침부 **위험단면 = 받침부 면**. 외부 받침부는 기둥머리/벽체 면에서 거리 ≤ **0.175·l1**.

## 5. 2방향 슬래브 배근 상세 (4.1.5)

### 5.1 소요철근량·간격 (4.1.5.1)
- 위험단면 휨모멘트로 결정, **최소 철근량 KDS 14 20 50(4.6) 이상**.
- 위험단면 철근 간격 **≤ min(2·t, 300 mm)** (와플/리브구조 예외).

### 5.2 정착 (4.1.5.2)
- 불연속 단부 직각방향 **정모멘트 철근**: 슬래브 끝까지 연장, 직선 또는 갈고리로 **≥ 150 mm** 테두리보/기둥/벽체에 묻음.
- 불연속 단부 직각방향 **부모멘트 철근**: KDS 14 20 52 따라 구부림/갈고리/기타로 받침부 면에서 테두리보/기둥/벽체 속 정착.
- 테두리보·벽체 미지지 불연속 단부나 캔틸레버: 슬래브 내부에 정착 가능.

### 5.3 외부 모퉁이 보강철근 (4.1.5.3)
**테두리보 α값 > 1.0**인 외부 모퉁이 슬래브:
- 상·하부에 단위폭당 **최대 정모멘트** 크기 휨모멘트 저항 철근 배치.
- 상부: 모퉁이 대각선에 **직각** 축에 대해 작용 가정.
- 하부: 모퉁이 대각선에 **평행** 축.
- **모퉁이에서 긴 경간의 1/5** 범위.
- 또는 슬래브 각 모서리에 평행하게 **두 층**으로 배치.

### 5.4 보 없는 슬래브 (플랫) 철근 상세 (4.1.5.4)

- 4.1.5 모든 요구 + **그림 4.1-1 최소 연장길이**.
- 인접 경간 길이 다르면 **긴 경간 기준**.
- 2방향 굽힘철근은 굽힘각 **≤ 45°** 가능한 경우만.
- 횡력 지지 2방향 슬래브는 구조해석으로 철근 길이 결정(최소 연장 이상).
- **각 방향 주열대 내 모든 하부 철근** 연속 또는 **A급 겹침이음**. 각 방향 **≥ 2개 주열대 하부철근**이 기둥 위를 지나며 외부 받침부 정착.
- 전단머리·리프트 슬래브: 각 방향 **2개 이상** 부착 하부 철근이 기둥 근접 위치로 전단머리/리프팅 칼라를 지나야 함.

**그림 4.1-1 요약 — 주열대/중간대 최소 철근 비율 (%)**

| 설계대 | 위치 | 지판 없음 | 지판 있음 |
| ------ | ---- | --------- | --------- |
| 주열대 | 상부 | 50% 나머지 | (상동) |
| 주열대 | 하부 | 100% | (상동) |
| 중간대 | 상부 | 100% | (상동) |
| 중간대 | 하부 | 50% 나머지 | (상동) |

## 6. 슬래브 시스템의 개구부 (4.1.6)

- 해석으로 검증 시 **어떤 크기의 개구부**도 가능.
- 해석 없으면 보 없는 슬래브에 아래 규정:
  - **중간대 겹치는 부분**: 원래 소요철근량 유지 시 모든 크기 개구부 허용.
  - **주열대 겹치는 부분**: 어느 경간에서나 주열대 폭의 **1/8 이상** 절단 금지. 절단된 철근량은 주변 추가 배치.
  - **주열대+중간대 겹치는 부분**: **1/4 이하** 철근만 절단. 주변 추가 배치.
  - **KDS 14 20 22(4.11.6)** 전단 규정 만족.
- 소규모 개구부: 절단된 철근과 **동일 단면적**을 개구부 양쪽에 보강.
- 대규모/치우친 개구부: 캔틸레버 또는 3변 연속 1변 자유 슬래브로 취급.

## 7. 기초판 설계 (4.2)

### 7.1 일반 (4.2.1)
- 계수하중과 반력으로 설계.
- 기초판 밑면적·말뚝 수·배열: **사용하중**(하중계수 미적용)과 토질역학 허용지지력으로 결정.
- 말뚝기초: 말뚝 반력이 각 말뚝 중심에 집중된다고 가정.
- 원형/정다각형 기둥: **같은 면적 정사각형**으로 위험단면 결정.
- **기초판 윗면 ~ 하부철근 깊이 ≥ 150 mm** (직접기초) / **≥ 300 mm** (말뚝기초).

### 7.2 휨모멘트에 대한 설계 (4.2.2.1)

**위험단면 위치**:
- 콘크리트 기둥/주각/벽체 지지: 기둥·주각·벽체 **외면**.
- 조적조 벽체 지지: 벽체 **중심과 단부의 중간**.
- 강재 밑판 기둥: 기둥 외측면과 강재 밑판 단부의 **중간**.

**철근 배치**:
- **1방향 또는 2방향 정사각형**: 전체 폭에 **균등** 배치.
- **2방향 직사각형**:
  - **장변방향** 철근 → 폭 전체 균등.
  - **단변방향** 철근 → 전체 중 식 (4.2-1) 비율만 **유효폭**(= 단변길이, 기둥/주각 중심선 = 중심)에 균등 배치, 나머지는 바깥에 균등.

### 7.3 전단력에 대한 설계 (4.2.2.2)
- 전단강도: **KDS 14 20 22(4.11)** (슬래브와 동일).
- 위험단면: 기둥/주각/벽체 외면(4.2.2.1(2)①) 기준. 강재 밑판은 (2)③.
- 말뚝 반력이 단면에 포함되는지 판단 (말뚝 중심 vs d/2):
  - 말뚝 중심이 단면에서 **d/2 이상 외측**: 전체 반력이 단면에 작용.
  - **d/2 이상 내측**: 전단력 작용 없음.
  - 중간: **직선보간**.

### 7.4 기초판 철근 정착 (4.2.2.3)
- KDS 14 20 52에 따라 정착.
- 위험단면 = 4.2.2.1(2) 최대 계수휨모멘트 위험단면 = 단면·철근 변경 수직면.

### 7.5 기둥 힘 전달 (4.2.3)
- 기둥/벽체 밑면에서 힘·모멘트는 콘크리트 **지압 + 철근/다웰/기계적 연결**로 기초판에 전달.
- 콘크리트 지압강도: 기둥 콘크리트 강도 한계 + 기초판 **KDS 14 20 20(4.7)** 지압강도.
- 강도 초과분은 철근·다웰·기계적 연결재가 담당.
- 현장치기(4.2.3.2): 다웰은 축방향 철근과 **겹침이음** 가능 (D35 이하 제한 등).
- 프리캐스트(4.2.3.3): 앵커볼트·기계적 연결재로 힘 전달.

### 7.6 특수 기초판 (4.2.4)
- **벽기초**(4.2.4.2): 벽체가 지지하는 1방향 기초판은 벽체와 직각방향 휨만 고려 가능.

## 9. 슬래브 설계 체크리스트

### 입력 / 유형 판별
- [ ] 장단변비 ≤ 2 (2방향) vs > 2 (1방향) 판정
- [ ] 보·지판 유무로 플랫/플랫 플레이트 판별
- [ ] fck, fy, 노출조건

### 두께·처짐
- [ ] 1방향 t ≥ 100 mm, 최소 두께 KDS 14 20 30(4.2.1) 준수
- [ ] 플랫 슬래브 지판 크기 (경간/6 이상, 두께 1/4 이상) 확인

### 해석법
- [ ] 직접설계법 8가지 조건 확인 → 아니면 등가골조/FEM
- [ ] Mo = wu·l2·ln²/8 계산 (ln ≥ 0.65·l1)
- [ ] 내부경간 부·정 = 0.65/0.35, 단부경간 표 4.1-1 적용

### 분배
- [ ] 주열대 내부 부 · 외부 부 · 정 모멘트 분배 (표 4.1-2/3/4)
- [ ] 보가 있으면 α1·l2/l1 따라 85% 내외 보 부담
- [ ] 중간대에 나머지 할당

### 전단
- [ ] 1방향 전단 (KDS 14 20 22 4.1~4.3)
- [ ] 2방향 뚫림전단 (KDS 14 20 22 4.11) — 별도 md 참조

### 배근
- [ ] 위험단면 철근 중심간격 ≤ min(2t, 300mm)
- [ ] 1방향 슬래브 주철근 직각방향 수축·온도철근
- [ ] 2방향 플랫 슬래브 그림 4.1-1 최소 연장 준수
- [ ] 각 방향 주열대 하부철근 ≥ 2개 기둥 위 통과

### 기초판
- [ ] 기초판 유효깊이 ≥ 150 mm(직기) / 300 mm(말뚝)
- [ ] 직사각 기초판 단변방향 철근 유효폭 내 식 (4.2-1) 비율 배치
- [ ] 뚫림전단 4.11 만족, 말뚝 반력 d/2 규칙 적용
