## 슬래브 설계 절차 개괄

```
┌──────────────────────────────────────────────────────────────┐
│ 1. 슬래브 유형 판별 (KDS 14 20 70 4.1.1 ~ 4.1.2)            │
│   - 장변/단변 ≤ 2 → 2방향, > 2 → 1방향                        │
│   - 보 유무 + 지판 유무 → 일반/플랫/플랫 플레이트             │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 2. 두께·처짐 제한 (KDS 14 20 70 4.1.1.3 / KDS 14 20 30)      │
│   - 1방향 슬래브 t ≥ 100 mm                                    │
│   - 최소 두께표(KDS 14 20 30(4.2.1))에 따른 t/l 비           │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 3. 해석·설계 방법 선정 (KDS 14 20 70 4.1.2.2)                │
│   - 직접설계법(4.1.3): 3경간 이상·장단변비 2 이하 등 8조건    │
│   - 등가골조법(4.1.4): 3D 골조 등가 해석                      │
│   - 일반 탄성/FEM 해석: 평형+적합조건                          │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 4. 휨 설계 + 모멘트 분배                                      │
│   - Mo = wu·l2·ln²/8 (전체 정적 계수휨모멘트)                  │
│   - 내부경간 부/정 = 0.65/0.35 Mo                              │
│   - 단부경간 표 4.1-1 (5가지 조건별)                           │
│   - 주열대/중간대 분배 (표 4.1-2~4)                            │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 5. 뚫림전단 설계 (KDS 14 20 22 4.11)                         │
│   - 위험단면 둘레 bo (기둥면에서 d/2)                          │
│   - Vc 식 (4.11-1) + bo·d                                      │
│   - 필요 시 스터럽/전단머리/확대머리 스터드                    │
│   - 불균형모멘트 편심전단 (4.11.7)                             │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 6. 배근 상세 (KDS 14 20 70 4.1.5)                            │
│   - 위험단면 철근 중심간격 ≤ min(2t, 300mm)                    │
│   - 기타 단면 ≤ min(3t, 450mm)                                 │
│   - 최소 수축·온도철근비 0.002 (fy 400MPa 이하 이형철근)      │
│   - 1방향 슬래브: 주철근 직각방향 수축·온도철근                │
│   - 보 없는 슬래브: 최소 연장길이(그림 4.1-1)                 │
└───────────────────────────┬──────────────────────────────────┘
                            ▼
┌──────────────────────────────────────────────────────────────┐
│ 7. 내진상세 (KDS 14 20 80 4.3.6) — 지진하중 적용 시          │
│   - 주열대 유효폭: 1.5h 이내 / 2h 이내                         │
│   - 받침부 주열대 상부철근 ≥ 1/4 전 경간 연속                  │
│   - 주열대 하부철근 ≥ 상부철근 1/3                             │
│   - 뚫림전단응력 ≤ 0.5·φVc (불균형모멘트 조건부)              │
└──────────────────────────────────────────────────────────────┘
```

## 슬래브 유형 정의

| 유형 | 정의 |
| ---- | ---- |
| 1방향 슬래브 | 마주보는 두 변에만 지지 / 2방향이어도 장단변비 > 2 |
| 2방향 슬래브 (보 있는) | 4변 모두 보 또는 벽체로 지지 |
| **플랫 슬래브** | 보 없이 기둥만으로 지지, **지판(drop panel) 있음** |
| 플랫 플레이트 | 보·지판 모두 없음, 기둥만 지지 |
| 와플 슬래브 | 2방향 리브로 조직된 격자형 슬래브 |
| 1방향 장선구조 | 1방향 리브 (KDS 14 20 10(4.11)) |
