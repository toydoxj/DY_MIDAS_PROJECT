## 2. 슬래브 철근 간격 (4.2.2(5))

- **벽체 또는 슬래브에서 휨 주철근의 간격**은
  - **슬래브 두께의 3배 이하**, **또한 450mm 이하**.
  - 단, 콘크리트 장선구조는 이 규정 적용 안 함.
- 2방향 슬래브 **위험단면**에서는 KDS 14 20 70(4.1.5.1)에 따라 **2·t, 300mm 이하** (더 엄격).

**순간격 일반 원칙** (4.2.2(1)(2)):
- 수평 평행철근 순간격 ≥ **max(25mm, db, 4/3·굵은골재 최대치수)**.
- 2단 배치 시 상하 순간격 ≥ 25mm, 상하 동일 연직면.

## 3. 슬래브 1방향 — 수축·온도철근 (4.6)

### 4.6.1 일반 원칙
- 슬래브에서 **휨철근이 1방향으로만 배치되는 경우**, 휨철근 **직각방향**으로 수축·온도철근 배치.
- 수축·온도 변형이 심하게 구속된 부재는 추가 최소 철근량 증가 필요.

### 4.6.2 1방향 RC 슬래브

| 항목 | 규정 |
| ---- | ---- |
| **최소 철근비 (이형철근/용접철망)** — fy ≤ 400 MPa | **0.0020** |
| **최소 철근비** — fy > 400 MPa | `max(0.0015·(400/fy), 0.0014)` (어떤 경우도 ≥ 0.0014) |
| 상한 | 단위 폭 m당 **1,800 mm² 이상 배치 불필요** |
| **간격** | 슬래브 두께의 **5배 이하, 450mm 이하** |
| 정착 | fy를 발휘할 수 있도록 정착 |

### 4.6.3 1방향 PSC 슬래브

- 유효프리스트레스로 콘크리트 전체 단면에 **평균 압축응력 ≥ 0.7 MPa** 발생시키도록 긴장재 배치.
- **긴장재 간격 ≤ 1.8 m**.
- 긴장재 간격이 **1.3 m 초과**시 4.6.2에 따른 수축·온도철근 추가 배치. 슬래브 단부에서 내측으로 긴장재 간격만큼 연장.

## 4. 2방향 슬래브 배근 상세 (참조 — KDS 14 20 70(4.1.5))

- 정착·최소 연장은 **KDS 14 20 70(4.1.5)** 및 **KDS 14 20 52(4.4)**.
- 슬래브 위험단면 철근 중심 간격 **≤ min(2·t, 300mm)**.
- 기타 단면 **≤ min(3·t, 450mm)**.
- 주열대 내 하부철근 연속성 및 A급 겹침이음 요구.

## 5. 구조 일체성 — 2방향 슬래브 (4.7.1(6))

- 2방향 슬래브 구조는 **KDS 14 20 70(4.1.5.4(5))** 규정에 따라야 함.
- 프리스트레스트콘크리트 2방향 슬래브는 **KDS 14 20 60(4.7.2(6), (7))** 규정.

## 6. 철근 허용오차 (4.2.1, 표 4.2-1)

| 유효깊이 d | 허용범위 | 최소 피복두께 |
| ---------- | -------- | ------------- |
| ≤ 200mm | ±10mm | −10mm |
| > 200mm | ±13mm | −13mm |

- 하단 거푸집까지 순거리 허용오차는 **−7mm**.
- 피복두께 오차는 요구 최소 피복두께의 **1/3 이내**.
- 슬래브 철근 종방향 구부림·단부 허용오차: **±50mm**.
- 부재 불연속단: **±25mm**.

## 7. 용접철망(WWF) 관련 (4.2.1(3))

- 경간 ≤ 3.0 m인 슬래브에 사용되는 **지름 6.4 mm 이하 용접철망**이 받침부를 지나 연속이거나 확실히 정착된 경우, **받침부 상단 부근 → 경간 중앙 슬래브 바닥 부근**으로 구부려 사용 가능.

## 8. 슬래브 철근 상세 체크리스트

### 피복
- [ ] 일반 슬래브: D35 이하 20mm / D35 초과 40mm (옥내)
- [ ] 흙 접촉 D19 이상 50mm / D16 이하 40mm
- [ ] ES 노출범주 슬래브 50mm 이상
- [ ] PSC 슬래브 옥외·흙 접촉 30mm

### 간격
- [ ] 위험단면 주철근 중심간격 ≤ min(2·t, 300mm)
- [ ] 기타 단면 ≤ min(3·t, 450mm)
- [ ] 수평 순간격 ≥ max(25mm, db, 4/3·굵은골재)

### 1방향 슬래브 수축·온도
- [ ] 주철근 직각방향 ρ ≥ 0.0020 (fy ≤ 400 MPa)
- [ ] 간격 ≤ min(5·t, 450mm)
- [ ] 항복강도 발휘 정착

### 2방향 슬래브
- [ ] KDS 14 20 70(4.1.5) 배근 준수, 그림 4.1-1 최소 연장
- [ ] 주열대 하부철근 A급 겹침, 각 방향 2개 이상 기둥 위 통과
- [ ] 외부 모퉁이 보강(α > 1인 테두리보) 배치
