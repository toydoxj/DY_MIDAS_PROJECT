# 슬래브 설계 절차 (KDS 종합)

> 실무에서 RC 슬래브(1방향·2방향·플랫)를 설계할 때 따라야 할 단계별 절차서. 각 단계에서 참조해야 할 KDS 조항 번호를 병기하며, 상세 식·표는 `01~04_*.md` 개별 문서 및 `../raw/` 원문을 참조하도록 한다.

## 설계 흐름 한눈에 보기

```
[0] 입력 정리 → [1] 유형 판별 → [2] 최소 두께·처짐 제어
  → [3] 해석법 선정 → [4] 휨모멘트 산정 → [5] 휨철근 설계
  → [6] 1방향 전단 → [7] 2방향 뚫림전단 → [8] 편심전단(Mu_unb)
  → [9] 사용성(균열·처짐 재검토) → [10] 배근 상세·정착·이음
  → [11] 개구부 처리 → [12] 내진 상세(해당 시) → 도면화
```

각 단계는 앞선 단계의 결과가 제한사항을 만족하지 않으면 단면·배근을 조정해 **루프**한다. 특히 **처짐 → 두께**, **뚫림전단 → 두께/주철근비/전단보강**, **편심전단 → 철근 집중배치/전단머리** 경로에서 반복이 많다.

## [0] 입력 조건 정리

| 항목 | 내용 |
| ---- | ---- |
| 재료 | fck (콘크리트 설계기준압축강도), fy (주철근), fyt (전단철근), Ec (= KDS 14 20 10(4.3.3)) |
| 단면 | 초기 가정 두께 t, 유효깊이 d = t − cc − db/2 (1방향) 또는 (t − cc − 1.5·db) (2방향, 양방향 평균) |
| 기둥 / 지지 | 기둥 치수 c1 × c2, 기둥 간격 l1, l2, 내부/외부/모서리 구분 |
| 하중 | 고정하중 D, 활하중 L, 마감 · 간막이, 풍 · 지진 E, 하중조합 (KDS 14 20 01 또는 시설물별 기준) |
| 환경 | 노출범주 (EC, ES 등), 내화 요구 두께, 습윤/해양 환경 여부 |
| 사용성 요구 | 처짐 제한 (표 4.2-2), 허용균열폭 (노출환경별) |

> **단위 일관성**: 모든 식은 **MPa (N/mm²) · mm · N** 기준. l, d, b는 mm.

## [1] 슬래브 유형 판별 및 지지조건

**KDS 14 20 70 4.1.1 ~ 4.1.2**

### 장변/단변비로 1차 분류

```
  장변 / 단변 ≤ 2 → 2방향 슬래브 (양방향 휨 분담)
  장변 / 단변 >  2 → 1방향 슬래브 (단변방향만 설계)
```

### 지지 조건으로 상세 분류

| 조건 | 유형 | 해석·설계 기본 |
| ---- | ---- | -------------- |
| 마주 두 변만 지지 | 1방향 슬래브 | KDS 14 20 20 보 설계법 |
| 4변 보 지지, α > 0.2 | 2방향 보 있는 슬래브 | 직접설계법 또는 등가골조법 |
| 4변 보 지지, α ≤ 0.2 | 2방향 보 없는 슬래브 | 플랫 슬래브/플랫 플레이트 처리 |
| 기둥만 지지 + 지판 | **플랫 슬래브** | 지판 설계 4.1.2.4 |
| 기둥만 지지 + 지판 없음 | **플랫 플레이트** | 두께·뚫림전단 더 엄격 |
| 2방향 리브 격자 | 와플 슬래브 | 등가균일판 치환 후 설계 |

α = 보 강성비 = E·Ib / (E·Is), 슬래브 보 강성 대비.

## [2] 최소 두께 및 처짐 제어

**KDS 14 20 30 (4.2)**

### 2.1 1방향 슬래브 최소 두께 (표 4.2-1)

비구조 요소 손상 우려 없는 경우, 처짐 계산을 생략할 수 있는 **최소 두께 t_min = l / r**:

| 지지조건 | 1방향 슬래브 | 보·리브 1방향 슬래브 |
| -------- | ------------- | -------------------- |
| 단순 지지 | **l / 20** | l / 16 |
| 1단 연속 | **l / 24** | l / 18.5 |
| 양단 연속 | **l / 28** | l / 21 |
| 캔틸레버 | **l / 10** | l / 8 |

보정:
- **경량콘크리트 (1,500~2,000 kg/m³)**: × `(1.65 − 0.00031·wc) ≥ 1.09`
- **fy ≠ 400 MPa**: × `(0.43 + fy/700)`

### 2.2 2방향 슬래브 최소 두께 (보 α ≤ 0.2인 경우, 표 4.2-3)

| fy (MPa) | 지판 없음 · 외부 (테두리보 없음 / 있음) | 지판 없음 · 내부 | 지판 있음 · 외부 (테두리보 없음 / 있음) | 지판 있음 · 내부 |
| -------- | --- | --- | --- | --- |
| 300 | ln/32, ln/35 | ln/35 | ln/35, ln/39 | ln/39 |
| 350 | ln/31, ln/34 | ln/34 | ln/34, ln/37.5 | ln/37.5 |
| **400** | **ln/30, ln/33** | **ln/33** | **ln/33, ln/36** | **ln/36** |
| 500 | ln/28, ln/31 | ln/31 | ln/31, ln/33 | ln/33 |
| 600 | ln/26, ln/29 | ln/29 | ln/29, ln/31 | ln/31 |

**하한**:
- 지판 **없음** → t ≥ **120 mm**
- 지판 **있음** → t ≥ **100 mm**

### 2.3 2방향 슬래브 — 보가 있는 경우 (α > 0.2)

- **0.2 < α_m < 2.0**: 식 (4.2-5) 이상, 또한 **≥ 120 mm**.
- **α_m ≥ 2.0**: 식 (4.2-6) 이상, 또한 **≥ 90 mm**.
- 불연속단: α ≥ 0.8인 테두리보가 없으면 위 두께의 **10% 이상 증가**.

### 2.4 처짐 계산 (두께 미만족 시)

1. 유효 단면2차모멘트 Ie (식 4.2-1): `Ie = (Mcr/Ma)³·Ig + (1 − (Mcr/Ma)³)·Icr ≤ Ig`
2. 순간처짐 Δi = 탄성처짐공식으로 Ie 적용.
3. 장기처짐 배수 λΔ = ξ / (1 + 50·ρ') (식 4.2-4). ξ: 5년↑ 2.0 / 12개월 1.4 / 6개월 1.2 / 3개월 1.0.
4. 총 장기처짐 = Δi + λΔ·Δsus.

### 2.5 최대 허용처짐 (표 4.2-2)

| 구조 | 처짐 | 한계 |
| ---- | ---- | ---- |
| 비구조 요소 미부착 평지붕 | L에 의한 순간처짐 | **l/180** |
| 비구조 요소 미부착 바닥 | L에 의한 순간처짐 | **l/360** |
| 비구조 요소 **부착** 지붕·바닥 | 부착 후 장기처짐 + L 순간처짐 | **l/480** |
| 손상 염려 없는 비구조 요소 부착 | 상동 | **l/240** |

동하중 구조:
- 단순·연속 활하중+충격 처짐 ≤ **l/800** (도시 인도겸용 l/1000)
- 캔틸레버 ≤ **l/300** (보행 고려 l/375)

> **결정**: 두께가 표 4.2-1, 4.2-3 만족 시 처짐 계산 생략 가능 (단, 표 4.2-2 사용성 제약이 있으면 계산으로 확인).

## [3] 해석법 선정

**KDS 14 20 70 4.1.2.2**

평형조건 + 기하학적 적합조건을 만족하는 어떤 방법이든 가능. 실무 선택지:

| 방법 | 적용 조건 | 참고 |
| ---- | --------- | ---- |
| **직접설계법 (DDM)** | 4.1.3.1의 8개 조건 **모두** 만족 | 4.1.3 |
| **등가골조법 (EFM)** | DDM 외 모든 경우, 격간 형태 복잡 | 4.1.4 |
| **탄성판 해석 / FEM** | 불규칙 형상, 개구부 많음, 비정형 지지 | 4.1.2.2(1) |
| **항복선 해석** | 극한하중·소성해석 필요 시 | (참고: KDS 14 20 10) |
| **1방향 슬래브 연속보 해석** | 장단변비 > 2 | 4.1.1.2 |

### 3.1 직접설계법(DDM) 8개 조건

1. 각 방향 **3경간 이상** 연속
2. 슬래브 판은 **장단변비 ≤ 2** 직사각형
3. 연속 경간 차이 ≤ 긴 경간의 **1/3**
4. 기둥 중심선 어긋남 ≤ 경간의 **10%**
5. **등분포 연직하중**, L ≤ 2·D
6. 보가 있는 경우 두 방향 보의 상대강성 식 (4.1-1) 만족
7. 휨모멘트 재분배 불가 (단, 4.1.3.7의 ±10% 수정은 가능)

## [4] 휨모멘트 산정

### 4.1 1방향 슬래브 (4.1.1.2)

연속보로 가정한 탄성해석 또는 KDS 14 20 10(4.1) 근사법. 보정:
- 활하중 **경간 중앙 부모멘트**는 계산값의 **1/2**만 취함.
- **경간 중앙 정모멘트**는 **양단 고정**으로 계산한 값 이상.
- 순경간 > 3.0 m: 순경간 내면 휨모멘트 사용 (고정단 이상).

### 4.2 2방향 — 직접설계법 (4.1.3)

#### 전체 정적 계수휨모멘트 Mo (식 4.1-2)

```
Mo = wu · l2 · ln² / 8
```
- wu: 계수하중, l2: 직각방향 경간, ln: 순경간 (≥ 0.65·l1).

#### 경간 분배

**내부 경간**: 부 0.65·Mo / 정 0.35·Mo.

**단부 경간 (표 4.1-1)** — 조건별 5가지:

| 위치 | (1) 구속無 | (2) 모든 받침 보 | (3) 내부 보無, 테두리보無 | (4) 테두리보有 | (5) 완전구속 |
| ---- | --------- | --------------- | ------------------------ | -------------- | ------------ |
| 내부 부 | 0.75 | 0.70 | 0.70 | 0.70 | 0.65 |
| 정 | 0.63 | 0.57 | 0.52 | 0.50 | 0.35 |
| 외부 부 | 0 | 0.16 | 0.26 | 0.30 | 0.65 |

#### 주열대(Column Strip) 분배 — 표 4.1-2,3,4 (`01_KDS_14_20_70_*.md` 참조)

핵심 값 (α1·l2/l1 = 0 기준, 단변방향 l2/l1 = 1.0):
- 내부 받침부 부모멘트: **75%** 주열대 / 25% 중간대
- 정모멘트: **60%** 주열대 / 40% 중간대 (α=0), 75% 주열대 (α·l2/l1 ≥ 1.0)

#### 중간대 (Middle Strip)
- 주열대가 부담하지 않는 정·부모멘트 잔여분을 양쪽 1/2 중간대에 비례 할당.

#### 보가 있는 경우 (4.1.3.5)
- α1·l2/l1 ≥ 1.0: 보가 **주열대 휨모멘트의 85%** 부담.
- 0 ~ 1.0 직선보간.

#### 모멘트 ±10% 수정 허용 (4.1.3.7)
- Mo 이상 유지 조건하에서 정·부 재분배 가능.

### 4.3 2방향 — 등가골조법 (4.1.4)

- 건물을 기둥 열마다 2D 등가골조로 분리.
- 활하중 배치: 각 경간 **3/4·활하중** + **인접경간 케이스별**.
- 내부 받침부 위험단면 = **받침부 면**. 외부 받침부는 기둥머리/벽체 면에서 ≤ **0.175·l1**.

### 4.4 FEM
- 유한요소 슬래브 모델 → 요소별 Mxx, Myy, Mxy 추출.
- 실무에서 **Wood-Armer** 식으로 직교방향 설계모멘트로 변환.

## [5] 휨철근 설계

### 5.1 소요 철근량 산정

강도설계 (KDS 14 20 20 4.1):
```
Mu ≤ φ·Mn = φ·As·fy·(d − a/2),   a = As·fy / (η·0.85·fck·b)
```
- 단위폭 b = 1,000 mm 사용 (슬래브는 mm²/m 으로 표시).
- 등가응력블록 계수 η, β1은 fck별 표 4.1-2 (KDS 14 20 20).

### 5.2 인장지배단면 확인 (KDS 14 20 20 4.1.2)

- 슬래브는 일반적으로 철근비가 작아 **인장지배**이지만 반드시 εt ≥ 0.005 (또는 fy > 400MPa이면 2.5·εy)을 확인.
- Pu < 0.10·fck·Ag 이면 축력 무시 가능.

### 5.3 최소 철근량 (KDS 14 20 50 4.6 / KDS 14 20 20 4.2.2(3))

| 구분 | 최소량 |
| ---- | ------ |
| **수축·온도철근비** (fy ≤ 400 MPa 이형철근) | **ρ ≥ 0.0020** |
| fy > 400 MPa | `ρ ≥ max(0.0015·400/fy, 0.0014)` |
| 상한 (필요량) | m당 1,800 mm² 이상 배치 불필요 |

슬래브 위험단면 휨철근은 이 값 이상 **및** 해석상 요구량 이상.

### 5.4 분배 (주열대 / 중간대)

직접설계법: 4.2의 주열대/중간대/보 분담률에 따라 계산된 M을 단위폭당 휨철근으로 환산해 **주열대 내 집중 배근**.

플랫 슬래브 보 없음 시 — 기둥 주변 주열대 내 **유효폭 (c2 + 3·h)** 내에 집중 (내진 4.3.6(3) 참조).

### 5.5 T형보 일체 플랜지 (4.1.2.1(4))
- 보와 일체로 된 슬래브: 보 양측 연장 ≤ **4·t**, 유효플랜지폭 계산 시 포함.

## [6] 1방향 전단 검토

**KDS 14 20 22 4.2, 4.11.1**

### 6.1 위험단면
- 받침부 내면에서 **d** 거리의 단면.

### 6.2 콘크리트 전단강도 Vc
```
Vc = (1/6) · λ · √fck · bw · d   (식 4.2-1)
```
- bw: 슬래브 폭 (단위폭 1,000 mm 사용).
- 필요 시 축력/휨에 의한 정밀식 (4.2-3).

### 6.3 검증
```
Vu ≤ φ·Vc
```
- 대부분 슬래브는 전단철근 없이 Vc로 충분 → **만족 못하면 두께 증가**.
- √fck ≤ **8.4 MPa** 제한 유지.

### 6.4 최소 전단철근 예외 (4.3.3(1)①, ③)
- 슬래브/기초판, 전체 깊이 ≤ 250 mm 보, 장선구조 등은 최소 전단철근 불필요.

## [7] 2방향 뚫림전단 검토

**KDS 14 20 22 4.11 — 슬래브 설계의 핵심 리스크**

### 7.1 위험단면 bo
- 기둥(기둥머리/지판) 면에서 **d/2** 만큼 떨어진 둘레.
- 둘레 최소가 되도록, 개구부 있으면 4.11.6 무효처리.
- 기둥 형상: **사각형은 네 변 평행 직선**, 원형·정다각형은 등가 정사각형.

### 7.2 콘크리트 뚫림전단강도 Vc

**RC (식 4.11-1, 4.11-2)**:
```
Vc = vc · bo · d
```
- vc는 식 (4.11-2)의 **이단 합** 형태 (압축대 인장응력 + 압축응력 성분).
- **ks (두께계수)** = (300/d)^0.25 ≥ 0.75 (식 4.11-3)
- **kbo** = 내부 1.0 / 외부(비모서리) 1.33 / 모서리 2.0 (식 4.11-4)
- 주인장철근비 ρ는 각방향 평균, **ρ ≤ 0.005면 0.005** 사용, **상한 0.03**.
- 주인장철근은 위험단면에서 bo 영역 + ld 이상 슬래브에 정착.

**PSC (식 4.11-9)** — 기둥 불연속단에서 4·t 이상, fck ≤ 35 MPa, 0.88 ≤ fpc ≤ 3.5 MPa 조건 만족 시 적용.

### 7.3 상한
```
Vn ≤ (2/3) · √fck · bo · d
```

### 7.4 Vu 산정 시 등분포 무시 가능 영역
- **슬래브-기둥 접합부**: 기둥면에서 0.5·d 내 등분포하중 무시 가능.
- **기초판-기둥 접합부**: 기둥면에서 0.75·d 내 등분포 지반력 무시 가능.

### 7.5 Vu > φVc인 경우 전단보강

#### 옵션 A — 스터럽
- 적용 조건: **d ≥ 150 mm**, db ≤ d/16.
- `Vs = Av·fyt/s`, **fyt ≤ 400 MPa**.
- **Vn ≤ (5/6)·√fck·bo·d**.
- 간격: 첫 열 **d/2**, 열 간 **d/2**, 기둥면 평행 다리 **2·d**.

#### 옵션 B — 전단머리 (I 또는 ㄷ형강)
- 4.11.4의 10개 상세 조건.
- αv ≥ 0.15, Mp 산정, 위험단면 재정의.

#### 옵션 C — 확대머리 전단스터드
- 4.11.5. 수직 배치, 첫 열 간격 ≤ d/2.
- PSC는 열 간 ≤ 0.75·d, 일반 RC는 ≤ 0.5·d.

#### 옵션 D — 지판(drop panel)/기둥머리
- 위험단면 확장으로 bo 증가 → Vu/bo·d 감소.
- 4.1.2.4의 지판 크기 규정: 경간의 **1/6 이상** 연장, **t 의 1/4 이상** 돌출.

> 전단보강 추가해도 부족하면 **두께 증가**로 귀결.

## [8] 불균형 휨모멘트 편심전단 검토

**KDS 14 20 22 4.11.7** — 연직/풍/지진하중으로 슬래브-기둥 모멘트 전달이 있는 경우 필수.

### 8.1 모멘트 분담

- **γf** = 휨으로 전달되는 분담 → 기둥면 양쪽 **0.5·t 폭** 주열대 내 휨철근 집중.
- **γv = 1 − γf** = 편심전단으로 전달.
- γf 공식은 KDS 14 20 22(4.11.7) + KDS 14 20 70(4.1.2.3(2)) 참조.

### 8.2 설계 검증 (식 4.11-13)
```
Mu ≤ φ·Mn_v = φ·(Mf + Mef + Mes)
```
- Mf: 전면+후면 슬래브 휨강도 합 (ρ ≤ 0.03 제한).
- Mef: 전면·후면 편심전단에 의한 휨강도 (식 4.11-16).
- Mes: 측면 편심전단 비틀림강도 (식 4.11-17).

### 8.3 양방향 불균형모멘트 (식 4.11-18)
```
(Mu,1 / φMn,1)^α + (Mu,2 / φMn,2)^α ≤ 1.0
```

### 8.4 전단머리와 연계 (4.11.7(6))
- 연직하중 전단응력 + 편심 전단응력 합 ≤ (1/3)·√fck.

### 10.5 인장정착길이 ld (KDS 14 20 52 4.1.2)

기본식 (식 4.1-1):
```
ldb = 0.6·db·fy / (λ·√fck)
```
상부철근 위치계수 α = 1.3 (상부·하부 300 mm 초과 생콘크리트). 기타 보정계수는 원문 표 4.1-1.

### 10.6 표준갈고리 ldh (KDS 14 20 52 4.1.5)
```
lhb = 0.24·βr·fy·db / (λ·√fck)
```
- **ldh ≥ max(8·db, 150 mm)**.
- 피복 ≥ 70 mm일 때 0.7 감소, 3·db 이하 간격 띠철근 있으면 0.8.

### 10.7 이음 (KDS 14 20 52 4.5)
- **인장 겹침이음**:
  - **A급 (1.0·ld)**: 배치량 ≥ 소요량×2, 이음 철근량 ≤ 전체 1/2.
  - **B급 (1.3·ld)**: A급 외 모두. ≥ 300 mm.
- **D35 초과 겹침이음 불가**.
- 플랫 슬래브 주열대 하부철근: **A급 겹침이음** + 각 방향 **2개 이상 기둥 위 통과**.

### 10.8 보 없는 슬래브 최소 연장 (그림 4.1-1)

| 중간대 | 상부 | 100% 연속 |

### 10.9 외부 모퉁이 보강철근 (4.1.5.3)

### 플랫 슬래브 / 플랫 플레이트
1. 두께: 표 4.2-3 (보통 **ln/30 ~ ln/36**), 하한 100/120 mm
2. DDM or EFM, 지판 크기 4.1.2.4 (경간 1/6, t 1/4) 확인
3. Mo → 분배, 주열대에 주로 집중 배근
4. **뚫림전단** 상세 검토 (4.11 전체) — 필요 시 스터럽/스터드/전단머리/지판
5. 불균형모멘트 편심전단 (4.11.7) 지진/풍 고려
6. 그림 4.1-1 최소 연장, A급 겹침, 각 방향 2개 이상 기둥 위 통과
7. 내진 시 4.3.6(8) Vug ≤ 0.5·φVc 준수

### 기초판
1. 하중계수 미적용 사용하중으로 면적·말뚝 배열
2. 유효깊이 ≥ 150mm (직접), 300mm (말뚝)
3. 위험단면 (기둥 외면) 휨설계
4. 직사각 기초판 단변방향 식 (4.2-1) 유효폭 내 집중
5. 1방향 + 2방향 뚫림전단 (4.11) 모두 검토, 말뚝 반력 d/2 규칙
6. KDS 14 20 20(4.7) 지압강도 + 다웰·앵커 정착
